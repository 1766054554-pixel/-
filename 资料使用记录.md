# 资料使用记录（公开脱敏版）

本文件只保留公开发布时可展示的资料处理原则和课程覆盖摘要。原始私人路径、临时附件路径、OCR 中间文件、浏览器抓取记录和未校对日志不进入公开仓库。

## 公开课程样例

- 数据库：SQL、关系代数、范式、事务恢复、ER 作图和 2023A 整卷交互。
- 人工智能：课程总纲、参考习题、模型对比、公式计算、生成模型和强化学习。
- 软件工程：官方五类题型、CodeArts 画图、开卷定位、案例建模和往年题变式。
- 军事理论：闭卷背诵、数字/定义/简答骨架和召回闸门。
- 离散数学：逻辑基础、范式、谓词、证明骨架、公式手册和近三年真题交互卷。
- 马原：老师最后重点、纸质开卷索引、材料题第一段和题眼定位。
- 操作系统：三套往年题交互卷和考点闪卡。

## 内容整理原则

1. 公开页面只放已经整理成学习动作的内容：讲义、题眼、真题、解析、错题回炉。
2. 原始资料、私人笔记路径、未脱敏日志和临时抽取文件不放进发布版。
3. 真题类内容尽量保留考试题型和作答动作，但公开使用时需遵守课程资料和版权边界。
4. 每次新增课程或题库，应同步更新 `docs/source_notes.md` 和 `docs/content_quality_rubric.md` 所要求的校对状态。

## 发布前确认

运行：

```bash
npm run validate
```

检查通过后，再提交到 GitHub 或部署到 GitHub Pages。
