二、填空速记：考前必须能顺嘴说出

1. AI/ML/DL/生成式 AI 的层级关系

展开参考答案

人工智能是总目标；机器学习是实现 AI 的重要方法；深度学习是机器学习中的一类多层神经网络方法；生成式 AI强调生成文本、图像、音频等新内容，常依托深度学习模型。

2. 过拟合、欠拟合、正则化

展开参考答案

过拟合：训练误差低，测试/验证误差高，说明模型记住训练集细节但泛化差。

欠拟合：训练误差和测试误差都高，说明模型能力不足、特征不够或训练不充分。

正则化：在损失里加入参数复杂度惩罚，例如 L(w) = ||Y - Xw||² + λ||w||²，鼓励更平滑、更简单的模型。

3. 强化学习四件套

展开参考答案

强化学习的基本语言是 状态 state、动作 action、奖励 reward、策略 policy。MRP 只有状态转移和奖励，MDP 多了动作与策略：智能体要在状态下选择动作，进而影响转移和奖励。

三、小计算：无需计算器也要会写步骤

1. L2 正则化损失

已知真实值 Y=[3,5]，预测值 Xw=[2,6]，参数 w=[1,2]，λ=0.5。计算 ||Y-Xw||² + λ||w||²。

展开解析

误差是 [1,-1]，平方和是 1² + (-1)² = 2。参数平方和是 1² + 2² = 5，正则项是 0.5 × 5 = 2.5。总损失 2 + 2.5 = 4.5。

迁移：如果题目问 λ 小于 0 为什么不行，答：它会奖励参数变大，不能起到约束复杂度的作用。

2. Precision / Recall / F1

某分类器对正类预测中，TP=8, FP=2, FN=4。计算 Precision、Recall 和 F1。

展开解析

Precision = TP/(TP+FP)=8/(8+2)=0.8。

Recall = TP/(TP+FN)=8/(8+4)=2/3≈0.667。

F1 = 2PR/(P+R)≈2×0.8×0.667/(1.467)≈0.727。

题眼：Precision 问“预测为正的里面有多少真阳性”，Recall 问“真实正类里找回了多少”。

3. AdaBoost 第一轮权重

4 个样本初始权重都为 0.25。某弱分类器只错分 1 个样本，所以 ε=0.25。若 α=0.5 ln((1-ε)/ε)=0.5 ln3，归一化后错分样本权重是多少？正确样本各是多少？

展开解析

更新前每个权重是 0.25。错分样本乘 e^α=√3，正确样本乘 e^{-α}=1/√3。

归一化后，错分样本权重会变成 0.5，三个正确样本各为 1/6。

为什么：AdaBoost 下一轮要更关注上一轮被错分的样本。考场不一定让你算这么细，但必须先算加权错误率，不能只数错题个数。

4. Q-learning 一步更新

已知 Q(s,a)=2，α=0.5，r=1，γ=0.9，下一状态最大动作价值 max Q(s',a')=4。计算新 Q(s,a)。

展开解析

Q-learning 更新式是：

Q ← Q + α [r + γ max Q(s',a') - Q]

目标值是 1 + 0.9×4 = 4.6，TD 误差是 4.6 - 2 = 2.6，新值是 2 + 0.5×2.6 = 3.3。

区分 Sarsa：Sarsa 用实际下一动作 Q(s',a')，Q-learning 用最大值 max Q(s',a')。

四、简答模板：会写第一句，才能拿分

1. 比较符号主义、联结主义、行为主义

展开参考答案

符号主义：用符号、规则和逻辑推理表示知识。优点是解释性强、接近人类显式推理；不足是知识库和规则难以完备，面对开放环境很脆。

联结主义：从数据中学习，例如神经网络和深度学习。优点是能处理图像、语音、文本等复杂模式；不足是依赖数据和算力，可解释性较弱。

行为主义：强调智能体与环境交互，通过奖励和惩罚学习策略，例如强化学习。优点是能学习决策能力；不足是探索成本高、样本效率和安全性问题明显。

2. 监督学习和强化学习的反馈有什么不同？

展开参考答案

监督学习的反馈通常是明确标签，模型预测后能立刻和正确答案比较，损失函数直接告诉模型错多少。强化学习没有每一步的标准答案，智能体采取动作后从环境得到奖励，奖励可能稀疏或延迟，目标是最大化长期累计回报。

一句话：监督学习像老师批改答案，强化学习像在游戏里试错并根据长期得分改策略。

3. LDA 决策边界为什么是线性的？

展开参考答案

LDA 假设各类别样本服从高斯分布，并且不同类别共享相同协方差矩阵。进行分类时比较后验概率，取对数几率比后，由于协方差相同，关于 x 的二次项会相互抵消，只留下 w^T x + b 形式，因此决策边界是线性的。

考试句式：不需要完整推导矩阵，只要写出“共享协方差导致二次项抵消”这个核心理由。

4. 为什么单层感知机不能表示 XOR？如何用前馈网络解决？

展开参考答案

XOR 的正负样本在二维平面上不能用一条直线分开，所以单层感知机这种线性分类器无法表示。加入隐藏层后，前馈网络可以先用多个神经元构造中间特征，再通过非线性激活组合出非线性决策边界，从而表示 XOR。

补一句更稳：多层网络的核心价值不是“层数看起来多”，而是线性变换和非线性激活的组合提高了函数表达能力。

5. PCA、AE、VAE、GAN、Diffusion 怎么区分？

展开参考答案
模型第一句题眼
PCA线性降维，找最大方差方向。主成分、特征人脸、正交方向。
AE编码再解码，通过重构学习表示。encoder/decoder、重构误差。
VAE学习概率隐空间，可从隐变量采样生成。μ、σ、KL、重参数化。
GAN生成器和判别器对抗训练。真假判别、模式崩塌、训练不稳定。
Diffusion正向加噪，反向学习逐步去噪生成。噪声预测、多步采样、生成质量高。

五、综合题：题干混在一起时怎么拆

综合题 1：训练曲线 + 指标 + 改进建议

某二分类模型训练误差持续降低，验证误差先降低后升高；另一个模型训练误差和验证误差都很高。题目还给出正类 TP=8, FP=2, FN=4。请判断问题，计算指标，并给出改进建议。

展开参考答案

曲线判断：训练误差低而验证误差升高是过拟合；训练和验证误差都高是欠拟合或优化不足。

指标计算：Precision = 0.8，Recall = 2/3，F1 约 0.727。

建议：过拟合可用正则化、早停、数据增强、降低模型复杂度；欠拟合可增加模型容量、改进特征、训练更充分或调整学习率。

综合题套路：不要一上来只写“加正则化”。先说明证据，再给方案。

综合题 2：图像分类任务选模型

给你一个医学图像分类任务，标注样本较少，但有大量无标注图像。要求说明可选模型、训练策略和风险。

展开参考答案

任务层：这是图像分类，输出是离散类别，属于监督学习任务；样本少意味着过拟合风险高。

模型层：可用 CNN 或 ViT 作为图像特征提取模型。数据少时优先考虑预训练模型和迁移学习，而不是从零训练大模型。

训练层：可采用数据增强、正则化、早停、交叉验证；无标注图像可考虑半监督学习或自监督预训练。

风险层：医学数据可能类别不平衡、标注噪声、分布偏移，评价时不能只看 Accuracy，要关注 Recall、Precision、F1 等。

综合题 3：AlphaGo 不是只靠一个算法

参考习题问 AlphaGo 没有使用哪种 AI 方法。请把 AlphaGo 用到的关键方法、没有使用的典型方法，以及为什么它体现混合智能写清楚。

展开参考答案

AlphaGo 综合使用了深度神经网络、监督学习、强化学习和蒙特卡洛树搜索。策略网络/价值网络帮助评估局面和选择动作，强化学习通过自我对弈改进策略，蒙特卡洛树搜索用于在巨大搜索空间中做更有效的决策。

它不是主要依靠符号主义的传统逻辑推理，也不是手写完备规则库逐步证明最优棋步。它体现的是学习、搜索和决策的结合。

考场第一句：AlphaGo 的题眼是“学习 + 搜索 + 强化学习”，不是单纯逻辑推理。
