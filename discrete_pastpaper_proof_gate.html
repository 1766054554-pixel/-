1. 先分八个盒子：离散不是一团雾

多年卷面看着吓人，但常见题眼很稳定。你考试时不要先问“我会不会”，先问“它属于哪一盒”。分对盒子，至少第一步不会空白。

定义简答完备集、合式公式、语义、谓词公理系统、永真/可满足/永假。先写定义，再写关系或例子。
量词反例∀x(Q(x) ∨ R(x)) ? ∀xQ(x) ∨ ∀xR(x) 这类题常用两元素论域，让性质分摊到不同对象上。
语义推论Γ ⊨ Q 成立就证明“前提全真时结论真”；不成立就造“前提全真、结论假”的反例。
主范式主析取看真值为 1 的行，主合取看真值为 0 的行。别凭感觉化简，先列真值表最稳。
前束/Skolem先改名、去箭头、否定内移，再量词前移。Skolem 只看 存在量词左边已经出现的全称变量。
可靠/完备可靠性：Γ ⊢ Q ⇒ Γ ⊨ Q。完备性：Γ ⊨ Q ⇒ Γ ⊢ Q。方向千万别反。
公理证明先看题目能不能用演绎定理；若明确禁用，就只能列公理、前提、MP 或已给定定理。
关系/函数证明最大元唯一、双射复合这类题，先写定义，再假设两个对象，最后用反对称/单射/满射收口。
记忆句：离散考场第一反应不是“推到底”，而是“定义先落地，反例先造小模型，证明先写目标”。

3. 证明起手模板：考场先写第一句

证明题最怕空白。下面每个模板都来自多年卷反复出现的证明类型。你先在文本框里写第一句，再展开参考骨架。

模板 A｜语义推论：判断 Γ ⊨ Q 是否成立

展开参考骨架
证明成立设任意解释/赋值 v，使 Γ 中所有公式为真。根据前提逐步推出 Q 为真，所以 Γ ⊨ Q。
证明不成立给出一个解释/赋值 v，使所有前提为真，但结论 Q 为假。这样就直接否定语义推出。
常见扣分只说“显然成立”；没有说明前提全真结论真；反例没有同时检查所有前提。

模板 B｜量词反例：全称不能随便拆

展开参考骨架
典型题∀x(Q(x)∨R(x)) ⇒ ∀xQ(x)∨∀xR(x) 不成立。
小模型取 D={1,2}，令 Q(1)=真、Q(2)=假，R(1)=假、R(2)=真。每个元素都满足 Q 或 R，所以左边真；但不是所有元素都满足 Q，也不是所有元素都满足 R，所以右边假。
迁移遇到“同一个存在对象能否服务所有 y”时，也常用两元素论域或关系表反例。

模板 C｜主析取/主合取：从真值表读答案

展开参考骨架
主析取找公式为真的行。每一行写一个极小项，变量为 1 写原变量，变量为 0 写否定变量，再把这些极小项用 ∨ 连接。
主合取找公式为假的行。每一行写一个极大项，变量为 0 写原变量，变量为 1 写否定变量，再把这些极大项用 ∧ 连接。
常见扣分把主析取和主合取的 0/1 行反了；跳过真值表直接化简导致漏项。

模板 D｜前束/Skolem：长公式按工序拆

展开参考骨架
前束步骤变量改名 -> 去箭头和等价 -> 否定内移 -> 量词前移 -> 写出母式。
Skolem若 ∃x 左边没有 ∀ 变量，用常量 a；若左边有 ∀y,∀z，则用函数 f(y,z)。只依赖左边已经出现的全称变量。
常见扣分让 Skolem 函数依赖右边才出现的变量；没有先改名导致自由/约束变量混住。

模板 E｜可靠性/完备性：两个箭头方向背牢

展开参考骨架
可靠性Γ ⊢ Q ⇒ Γ ⊨ Q。证明系统不乱证明，能证明出来的东西语义上是真的。
完备性Γ ⊨ Q ⇒ Γ ⊢ Q。语义上必然成立的东西，证明系统也能证明出来。
常见扣分把两个方向反写；把“可靠”和“完备”误理解成系统越大越好。

模板 F｜最大元唯一：定义 + 反对称

展开参考骨架
定义b 是 B 中最大元：对任意 c∈B，都有 c≤b。
证明设 b1,b2 都是最大元。因为 b1∈B 且 b2 是最大元，所以 b1≤b2；因为 b2∈B 且 b1 是最大元，所以 b2≤b1。由偏序的反对称性，b1=b2。
迁移唯一性证明常用“假设两个 -> 相互推出 -> 用定义/性质合并”。
