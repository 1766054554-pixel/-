1. Day1 正确顺序：先打地基，再碰题

你不用一上来把所有页面都看完。按下面顺序走，累了就停在保底包；状态好再往后推进。

保底 1：数据库 SQL 从零 先看 数据库 SQL 从零深学讲义。目标不是写复杂 SQL，而是知道表、行、列、主键、外键、WHERE、GROUP BY、HAVING 各自在干什么。
保底 2：离散基础地图从零 看 离散基础地图从零深学讲义 的 0-6 节。目标是读懂命题、联结词、真值、谓词、量词、辖域和证明第一句，不碰整张卷。
标准 1：AI 从零 看 AI 从零深学讲义 的“三关路线”、术语表、训练流程和深度学习故事线。目标不是背缩写，而是知道题目在问哪类任务、模型为什么这样设计、公式符号在干什么。
标准 2：软工画图从零 看 软工画图从零深学讲义 的 Day1 过关线、四支笔、图链路和“第一笔怎么下”。目标是看到题干先判断图类型，而不是立刻画。
加餐：AI 参考习题桥接 主讲义看完还有力气，再看 AI 参考习题桥接讲义。它是把真实参考习题拆成题眼，不是直接刷整卷。
今天不要追求“页面全清空”。你最重要的产物是能闭眼讲出来。看完不复述，等于只获得熟悉感；熟悉感在考场上很容易失效，这就是你操作系统那次暴露出来的问题。

2. 掌握闸门：过了这些，才算 Day1 真学过

下面不是考试分数，是“能不能进入下一层”的闸门。如果某科没过，明天先补它，不急着开新坑。

科目C 级：刚开始B 级：可以进入 Day2A 级：可以加真题变式
数据库 能说出表、行、列、主键、外键是什么。 能按 FROM/JOIN -> WHERE -> GROUP BY -> HAVING -> SELECT 解释一条 SQL。 能把“每个部门最高工资员工”拆成分组求最高 + 回表找员工。
AI 能解释样本、特征、标签、模型、参数、损失。 能区分回归/分类/监督/无监督/强化学习，并说出题眼。 能用人话解释线性回归、SVM、决策树、Q-learning 的考试关键词。
软工 能说出用例图、顺序图、类图分别回答什么问题。 看到场景能先圈角色、动作、对象、状态，并判断图类型。 能按题眼建立开卷索引：include/extend、后置条件、系统顺序图、类图关系。
离散 能解释命题、联结词、∀、∃、⊧、⊢。 能说出蕴涵唯一为假、辖域怎么看、证明 A→B 第一行怎么写。 能做命题判定、蕴涵真假、全称/存在翻译、简单辖域判断，并说出证明目标的第一句。
最小通过线 数据库和离散至少到 B；AI 和软工至少到 C+。因为 6/22 数据库最近，离散又需要慢热。
不通过怎么办 不要自责，也不要刷题逃避。回讲义对应章节，把术语用自己的话重讲一遍，再做一题题眼训练。

3. 今天先别做这些：它们会制造“我学不会”的错觉

别一上来刷整卷 你现在知识记忆接近 0，整卷会变成凭感觉猜。真题要做，但要等基础词典和模板先建立。
别只点选择题 选择题很高效，但必须先写题眼和人话答案。否则你只是识别选项，不是在训练考场调用。
别追求一天学完离散证明 离散证明要分层：先会读公式，再会选证明第一句，最后再练完整严谨证明。
别把开卷当免学 软工和马原开卷都需要索引。开卷不是资料多，而是看到题眼知道翻哪里、套什么模板。

4. 明天怎么接：按你今天的闸门结果分流

今天完成情况明天第一件事然后再做
只完成数据库保底 补数据库 SQL 条件/聚合，做 2 道分组题。 AI 只看术语表；离散只复述命题/量词。
数据库 B，离散 C 打开 离散 Day2 证明入口，补辖域、⊧/⊢、证明第一句。 数据库进入 Day2 SQL 连接/嵌套加厚讲义；AI 看回归/分类。
数据库 B+，AI/软工/离散都有碰 先完成 数据库 Day2 SQL 连接/嵌套 的 6 张题卡。 AI 做参考习题桥接；软工整理图类型索引。
状态很好，四科 Day1 都过 B 如果 SQL Day2 已到 B+，再加数据库 2023A 后半：事务、BCNF、ER 图。 离散完成 证明入口与基础题卡，暂不分析整张 2024 卷。

这就是后面陪跑的规则：不是每天重来，而是看你过了哪个闸门，然后继续加深。
