0. 总故事线：先看数据形状，再选模型

AI 后半不是一堆互不相干的英文缩写。你可以先问一个很朴素的问题：我的输入长什么样，输出要什么东西？ 输入是图片，就要处理局部空间结构；输入是长序列，就要处理远距离依赖；输出要“凭空造样本”，就进入生成模型。

图像线
图片有局部空间结构，所以先学 CNN；网络太深不好训，就引入 ResNet 的跳跃连接；如果想用注意力看全局关系，就把图片切成 patch，变成 ViT。
文本线
文本有顺序和长距离依赖。RNN/LSTM 是按时间步读；Transformer 改成所有 token 直接互看。BERT 偏理解，GPT 偏生成。
生成线
生成模型的共同目标是学会数据分布，然后造新样本。AE 学重构，VAE 学连续潜空间，GAN 学对抗，Diffusion 学逐步去噪。

0.5 新词护栏：先知道它解决什么，再记名字

AI 后半最大的痛苦不是难算，而是新词太密。下面这组不是词典抄定义，而是考场分流护栏：每个词先回答“它为什么存在”，再给题眼、相邻易混项和第一句话。读后面正式内容前，先把这组卡扫一遍。

CNN

图片不是普通表格，附近像素会组成边缘、纹理、形状。CNN 就是让小窗口在图片上滑动，专门抓局部模式。

题眼
卷积核、局部窗口、特征图、池化、图像分类。
别混
ViT 是 patch/token/attention；CNN 是卷积核滑动。
第一句
CNN 利用局部感受野和参数共享提取图像局部特征。
小测：题干说“3x3 filter 扫过图片”，先写 CNN，不要写 Transformer。

ResNet

网络越深，训练越容易卡住。ResNet 用跳跃连接让输入直接绕过若干层，再和输出相加。

题眼
skip connection、F(x)+x、深层网络、梯度消失。
别混
它不是池化，也不是注意力；它主要解决深层网络训练问题。
第一句
ResNet 通过残差连接帮助信息和梯度跨层传播。
小测：看到 F(x)+x，先解释“残差”和“跳跃连接”。

ViT

Transformer 原本处理 token 序列。ViT 把图片切成 patch，让每个 patch 变成视觉 token，再用注意力看全局关系。

题眼
patch embedding、视觉 token、Transformer encoder。
别混
CNN 先局部卷积；ViT 先切 patch，再全局 attention。
第一句
ViT 将图像切成 patch 并转为 token，用 self-attention 建模全局关系。
小测：题干写“图片切块后输入 Transformer”，模型名先写 ViT。

Transformer / Attention

RNN 要一步步传信息，长距离关系容易丢。Attention 让任意位置直接互相看，Transformer 把它做成主结构。

题眼
Q/K/V、self-attention、长距离依赖、并行。
别混
Q/K/V 不是 Q-learning；这里 Q 是 Query，K 是 Key，V 是 Value。
第一句
Attention 用 Q 和 K 计算相关性权重，再按权重汇总 V。
小测：问 √d_k 时，答“防止点积过大导致 softmax 过尖”。

BERT / GPT

二者都来自 Transformer，但考试要你区分任务倾向：BERT 偏理解，GPT 偏生成。

题眼
encoder-only、decoder-only、双向上下文、自回归。
别混
BERT 是双向理解；GPT 是按前文预测下一个 token。
第一句
BERT 偏语言理解，GPT 偏自回归文本生成。
小测：看到“根据前文生成下一个词”，先写 GPT。

AE / VAE

AE 学压缩和重构，VAE 进一步让隐空间变成可采样分布，所以更像生成模型。

题眼
编码器、解码器、重构误差、μ/logσ²、KL、重参数化。
别混
普通 AE 不一定有连续可采样隐空间；VAE 有分布约束。
第一句
VAE 用 μ 和 σ 表示潜变量分布，并通过 KL 让隐空间可采样。
小测：题干出现 μ/logσ² 和 KL，先写 VAE，不写普通 AE。

GAN

GAN 用两个网络互相较劲：生成器造样本，判别器辨真假。它追求逼真，但训练可能不稳。

题眼
generator、discriminator、adversarial、mode collapse。
别混
VAE 是潜变量分布和 KL；Diffusion 是加噪/去噪。
第一句
GAN 通过生成器和判别器对抗训练，使生成样本逼近真实分布。
小测：问“模式崩塌”，要说生成器只覆盖少数样本模式。

Diffusion / DDPM

扩散模型把生成拆成许多小步：前向逐步加噪，反向学会预测噪声并去掉噪声。

题眼
加噪、去噪、x_t、时间步 t、预测噪声 ε。
别混
它不是 GAN 对抗，也不是 VAE 的 KL 采样。
第一句
扩散模型训练网络根据 x_t 和 t 预测噪声，再逐步去噪生成样本。
小测：问“网络输出目标”，优先答预测噪声 ε。
记忆法：图像线看“局部到全局”：CNN -> ResNet -> ViT；文本线看“顺序到注意力”：RNN/LSTM -> Transformer -> BERT/GPT；生成线看“训练信号”：重构 AE、分布 VAE、对抗 GAN、去噪 Diffusion。

1. 深度学习训练四件套：数据、模型、损失、优化器

看到任何深度学习题，先别被模型名吓到。考试最爱问的是：数据怎么分、模型怎么前向算、损失衡量什么、参数怎么更新。

词从零解释考试题眼易错点
训练集模型真正用来学习、更新参数的数据。train、fit、backprop、update。数据增强通常只放训练集。
验证集训练过程中用于调超参数、选模型。validation、调学习率/epoch。不能拿它反复当最终成绩。
测试集模型训练完成后，评估泛化能力。test、unseen、generalization。不能参与训练和调参。
前向传播输入一路算到输出，得到预测和 loss。forward、logits、prediction。不是更新参数的阶段。
反向传播从 loss 反向求梯度，告诉每个参数该怎么改。backward、链式法则、gradient。只要网络中间不可导，就会卡训练。
学习率每次参数更新迈多大步。learning rate、震荡、收敛慢。太大可能震荡，太小可能太慢。
数据增强为什么只给训练集 增强是为了让模型见更多变化、学会泛化。验证集和测试集是“模拟考/期末卷”，如果你也随机翻转、旋转，就相当于改考题，评估不客观。
交叉熵为什么常用于分类 分类关心正确类别概率。真实类别概率越低，交叉熵惩罚越大；真实类别概率越高，惩罚越小。
ReLU(x) = max(0,x) , L = − C∑c=1 yclog(pc)
读法：ReLU 引入非线性；交叉熵惩罚“真实类别的预测概率太低”。

2. CNN、ResNet、ViT：图像题从局部到全局

图像模型的主线是：局部特征先由 CNN 抓，深层网络用 ResNet 稳住训练，全局关系可以交给 ViT/Attention。

CNN 解决什么问题 图片相邻像素组成边缘、纹理、形状。卷积核像一个会学习的小窗口，在图片上滑动找同一种局部模式。
ResNet 解决什么问题 网络太深时，梯度传到前面会变弱。跳跃连接让输入直接绕过若干层，与输出相加，帮助深层网络训练。
ViT 解决什么问题 它把图像切成 patch，每个 patch 当成 token，用 Transformer 建模 patch 之间的全局关系。
y = F(x) + x
读法：ResNet 不要求某几层完全重新学出 y，而是学习残差 F(x)，再把输入 x 加回来，缓解深层网络训练困难。

3. Transformer、BERT、GPT：Q/K/V 是考试入口

Transformer 的核心不是“很大很高级”，而是把序列中的每个位置都变成可以互相查询的信息点。Q、K、V 是公式题和简答题最常见入口。

Attention(Q,K,V) = softmax( QKT√dk ) V
读法：Q 和 K 算相关性，softmax 变成注意力权重，再用权重汇总 V。除以 √d_k 是为了数值稳定。
Q：我想查什么 当前 token 发出查询。简答时可以写“Query 表示当前位置需要寻找的信息线索”。
K：我有什么索引 每个 token 提供 key，用来和 Q 算匹配程度。QK 越大，说明越相关。
V：真正拿来汇总的内容 softmax 后的权重不是直接输出，而是去加权求和 V，得到当前位置的新表示。
概念核心架构核心任务/能力考试一句话
TransformerEncoder-Decoder序列建模并行计算、全局依赖，是现代大模型基础。
BERTEncoder-only语言理解双向上下文，常见理解类任务。
GPTDecoder-only文本生成自回归生成，按前文预测下一个 token。
AgentLLM + 工具 + 规划自主任务执行大模型调用工具、规划步骤、闭环行动。
答题模板：先写 Transformer 的核心是 self-attention，再解释 Q/K/V 和 √d_k，最后按任务区分 BERT/GPT：BERT 偏双向理解，GPT 偏自回归生成。

4. 生成模型：从“还原”到“造新样本”

生成模型的共同问题是：模型怎么学会数据长什么样，然后从某种隐变量或噪声出发造出新样本。不要把 AE、VAE、GAN、Diffusion 混成一团，它们的训练信号完全不同。

模型先解决的问题训练信号期末高频问法
AE把输入压缩成隐表示再重构。重构误差，如 MSE/L1。为什么 AE 不适合作为真正生成模型？隐空间可能不连续，随机采样容易落到没学过区域。
VAE让隐空间可采样、连续。重构损失 + KL 散度。重参数化为什么存在？为了让采样过程可导，能反向传播。
GAN生成逼真样本。生成器和判别器对抗。为什么训练不稳定？可能判别器过强、梯度不稳、模式崩塌。
Diffusion通过逐步去噪生成高质量样本。预测噪声的误差。它预测的是噪声，不是一步直接预测清晰图；优点稳定，缺点采样慢。
z = μ + σ ⊙ ε , ε ∼ N(0,1)
VAE 重参数化读法：随机性放到 ε 里，μ 和 σ 仍然可由网络输出并参与反向传播。
Ldiffusion = E [ || ε − εθ(xt,t) ||2 ]
扩散模型读法：给带噪图 x_t 和时间步 t，网络学习预测这一步噪声 ε。

6. 题卡：先写题眼，再看解析

lab md / 图像分类 题 1：CNN 为什么比全连接更适合图像？

题干：某模型用于动物图片分类，结构中包含 Conv2D、MaxPool、ReLU 和 Linear。请解释 CNN 适合图像的原因，并说明 Conv2D、MaxPool、ReLU 的作用。

展开参考答案

题眼：图片分类、Conv2D、MaxPool、ReLU。

标准答案：图像有局部空间结构，相邻像素会形成边缘、纹理、角点等局部特征。Conv2D 使用可学习卷积核在局部窗口滑动，提取局部特征；同一个卷积核在整张图上共享参数，因此比全连接参数更少，也保留空间邻近关系。MaxPool 保留局部强响应并降低特征图尺寸，减少计算量并提高一定平移鲁棒性。ReLU 引入非线性，否则多层线性变换叠加仍等价于线性模型。

迁移：换成“为什么 CNN 能提取边缘纹理”，仍然按局部结构、卷积核、参数共享作答。

PPT 第7-9讲 / ViT 题 2：ViT 和 CNN 的直觉差异是什么？

题干：某模型先把图片切成固定大小的小块，再把每个小块映射成向量输入 Transformer。请说明这是哪类模型，并比较它和 CNN 的建模直觉。

展开参考答案

题眼：图片切块、patch、Transformer。

标准答案：这是 Vision Transformer。ViT 把图片切成 patch，把每个 patch 展平并映射成视觉 token，然后用 Transformer 的 self-attention 建模 patch 之间的全局关系。CNN 的直觉是局部卷积：先用卷积核提取边缘、纹理等局部特征，再逐层组合；ViT 的直觉是把图像转成 token 序列，直接通过 attention 学全局依赖。

排除：如果题干强调卷积核滑动、池化、特征图，优先是 CNN；强调 patch/token/attention，优先是 ViT。

Transformer PPT / 公式解释 题 3：解释 Attention 公式

题干：解释 Attention(Q,K,V)=softmax(QK^T/√d_k)V 中 Q、K、V 和 √d_k 的含义，并说明 Transformer 为什么适合长距离依赖。

展开参考答案

题眼：Attention、Q/K/V、√d_k、长距离依赖。

标准答案：Q 是查询，表示当前位置想寻找什么信息；K 是键或索引，用来和 Q 计算相关性；V 是真正被加权汇总的内容。QK^T 得到相关性分数，softmax 将分数归一化成注意力权重，再用权重汇总 V。除以 √d_k 是为了防止向量维度较大时点积过大，使 softmax 过尖、梯度不稳定。Transformer 中任意两个位置可以直接通过 self-attention 建立联系，不需要像 RNN 那样逐步传递隐藏状态，因此更适合长距离依赖，也更易并行。

课程总结 / 大模型 题 4：BERT、GPT、Agent 怎么区分？

题干：表格给出 Transformer、BERT、GPT、Agent，要求填写核心架构、任务能力和特点。请写出最稳的比较答案。

展开参考答案

标准答案：Transformer 是序列建模基础架构，典型是 Encoder-Decoder，特点是并行计算和全局依赖。BERT 是 Encoder-only，偏语言理解任务，强调双向上下文。GPT 是 Decoder-only，偏文本生成，强调自回归，即根据前文预测下一个 token。Agent 可以理解为 LLM 加工具调用和规划能力，用于自主任务执行、闭环行动。

常见坑：不要把 BERT/GPT 的 encoder-only、decoder-only 说反；不要只写“都是大模型”，这没有区分度。

生成模型 lab / VAE 题 5：VAE 为什么要重参数化？

题干：VAE 编码器输出 μ 和 logvar，需要从潜变量分布中采样 z。为什么要使用 z=μ+σ⊙ε 的重参数化技巧？KL 散度在 VAE 中有什么作用？

展开参考答案

题眼：μ、logvar、采样、reparameterization、KL。

标准答案：VAE 希望从潜变量分布中采样 z 再送入解码器，但直接随机采样会阻断梯度，网络无法通过反向传播学习 μ 和 σ。重参数化写成 z=μ+σ⊙ε，其中 ε 从标准正态分布采样，随机性被移到 ε，μ 和 σ 仍由网络计算并可求导。KL 散度用于约束编码器输出的潜变量分布接近标准正态分布，使潜空间更连续、更可采样，从而更适合生成。

迁移：如果题目问“VAE 为什么比 AE 更适合生成”，就写 AE 隐空间可能离散不连续，VAE 通过分布建模和 KL 约束让潜空间可采样。

参考习题 / GAN 题 6：GAN 损失小但生成质量差，可能为什么？

题干：训练 GAN 时，发现损失函数值很小，但生成器产生的数据质量并不高。请解释一个可能原因，并说明 GAN 训练为什么不能保证稳定收敛。

展开参考答案

题眼：GAN、损失小但质量差、训练不稳定。

标准答案：GAN 的生成器和判别器需要保持相对平衡。如果判别器过强或过拟合，它可能在训练集上损失很小，但给生成器的梯度信号不再能有效指导生成器改进，导致生成质量差。GAN 是生成器和判别器的对抗优化，目标非凸且双方互相影响，可能出现梯度不稳定、模式崩塌等问题，因此不能保证训练足够久就全局最优。

PPT 第10-11讲 / Diffusion 题 7：扩散模型到底预测什么？

题干：DDPM 前向过程逐步向图像加高斯噪声，反向过程用神经网络生成图像。请说明训练时网络的输入、输出目标和扩散模型相对 GAN 的一个优缺点。

展开参考答案

题眼：加噪、去噪、DDPM、预测噪声。

标准答案：扩散模型前向过程是固定的逐步加噪，将真实图像变成不同噪声程度的 x_t。训练反向网络时，输入通常是带噪图 x_t 和时间步 t，目标是预测这一步加入的噪声 ε，而不是一步直接预测清晰原图。生成时从纯噪声开始，反复预测并去掉噪声，逐步得到清晰样本。相对 GAN，扩散模型训练更稳定、生成质量高，但需要多步去噪，采样速度较慢。

综合比较 / 考前简答 题 8：一句题眼分流

题干：请把下面 4 个描述分别对应到 AE、VAE、GAN、Diffusion，并说明理由：A. 学会压缩和重构输入；B. 生成器和判别器对抗；C. 通过重参数化从连续潜空间采样；D. 训练网络预测噪声逐步去噪。

展开参考答案

标准答案：A 对应 AE，因为 AE 的核心是编码压缩和解码重构；B 对应 GAN，因为 GAN 的核心是生成器和判别器对抗训练；C 对应 VAE，因为 VAE 用 μ、σ 和重参数化从连续潜变量分布中采样；D 对应 Diffusion，因为扩散模型训练网络预测噪声，并在生成时逐步去噪。

迁移：这类题不是背全称，而是抓“训练目标”。重构是 AE，对抗是 GAN，分布采样是 VAE，预测噪声是 Diffusion。
