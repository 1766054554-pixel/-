综合题四套模板：先分流，再展开

曲线题先看训练损失和测试损失。训练低测试高 -> 过拟合；两者都高 -> 欠拟合；训练降不动 -> 优化问题。再写对应改进：正则化、数据增强、早停、调学习率、增减模型复杂度。
表格题先找 TP、FP、FN、TN 或各类样本权重。评价指标题按 Precision/Recall/F1；AdaBoost 题按加权错误率、alpha、权重更新、加权投票。
结构图题先说每个部件在解决什么问题。CNN 写卷积/池化/ReLU；Transformer 写 Q/K/V 和全局依赖；VAE 写 μ/logσ²、重参数化、KL；Diffusion 写加噪/去噪/预测噪声。
场景设计题先判断任务：回归/分类/聚类/降维/强化学习。再写数据、模型、损失或奖励、训练、评价，最后补一个风险点。

2. 题眼速查：看到这些词，马上切模型

连续数值预测回归。线性回归常用 MSE，目标是让预测值接近真实值。
二分类概率逻辑回归。sigmoid 输出概率，交叉熵常用于分类。
最大间隔/支持向量SVM。关键词是超平面、margin、support vector、kernel。
熵/信息增益决策树。选能让不确定性下降最多的特征。
错分样本权重增加AdaBoost。弱分类器加权组合，错的样本下轮更重要。
训练集好测试集差过拟合。解决思路：正则化、更多数据、简化模型、早停。
类别不平衡看 Precision/Recall/F1，不要只看 Accuracy。
误差反向传播BP。链式法则把输出误差往前传，更新各层参数。
长期依赖/门控LSTM。输入门、遗忘门、输出门控制信息保留。
Q/K/V、自注意力Transformer。Attention 用权重汇总相关位置的信息。
聚类中心/簇内平方误差K-means。分配到最近中心，再更新中心为均值。
状态/动作/奖励/策略强化学习。Q-learning 更新当前状态动作对。

3. 公式默写：不只背公式，要解释符号

公式/关键词考试要会说什么常见问法
J=1nn∑i=1(yi−ŷi)2 预测值和真实值差距的平方平均。回归题看到“连续值预测 + 最小二乘”先叫它。 线性回归损失函数是什么？为什么异常点会影响大？
w*=(XTRX)−1XTRY 加权最小二乘闭式解。X 是特征矩阵，Y 是标签，R 给样本不同权重。 解释公式中每个量；权重 ri 有什么作用？
σ(z)=11+e−z 把实数映射到 0-1 概率；逻辑回归名字叫回归，考试里常做二分类。 逻辑回归为什么能做二分类？为什么常配交叉熵？
margin=2||w|| SVM 找最大间隔超平面，支持向量是离边界最近、真正撑住边界的样本。 SVM 的核心思想？支持向量是什么？
H(D)=−K∑k=1pklog2pk 熵衡量不确定性；类别越混，熵越大，越纯，熵越小。 决策树如何衡量划分前后的混乱程度？
Gain=H(D)−∑v|Dv||D|H(Dv) 分裂后不确定性下降多少；步骤是总熵、子集熵、加权熵、相减。 信息增益小计算；为什么信息增益率要惩罚取值多？
Precision=TPTP+FP 预测为正的里面有多少真正。误报代价高时更关注它。 预测为正 30 个，真为正 24 个，怎么算？
Recall=TPTP+FN 真实正样本里找回多少。漏报代价高时更关注它。 真实正样本 40 个，找回 24 个，怎么算？
Attention=softmax(QKT√dk)V Q 查什么，K 用来匹配，V 是被汇总内容；缩放是为了稳定 softmax。 解释 Q/K/V；为什么除以 √dk？
Qnew=Qold+α[r+γmaxa′Q(s′,a′)−Qold] 旧 Q 往“即时奖励 + 折扣后的下一状态最好价值”靠近，只更新当前格。 Q-learning 一步更新；为什么它是 off-policy？

4. 模型比较：简答题按“定义-原理-优缺点-适用场景”写

模型核心一句话优点局限
线性回归用线性函数预测连续值。简单、可解释、计算方便。表达能力有限，难拟合复杂非线性。
逻辑回归线性打分后用 sigmoid 输出类别概率。简单、概率解释清楚。决策边界较简单。
SVM最大化分类间隔，支持核方法。小样本、高维场景表现好。大数据训练成本高，参数和核函数敏感。
决策树用一串特征判断做分类/回归。可解释、能处理非线性。容易过拟合。
AdaBoost把多个弱分类器按权重组合。能提升弱学习器效果。对噪声和异常点敏感。
CNN卷积核在局部窗口滑动，提取边缘、纹理等局部特征。适合图像，参数共享减少参数量。天然全局关系弱，尺度/旋转变化要靠数据增强等补。
ViT把图像切成 patch，当作视觉 token 交给 Transformer。能建模全局关系。通常更依赖数据量和训练成本。
RNN/LSTM处理序列，LSTM 用门控缓解长期依赖。适合文本、时间序列。训练慢，长序列仍可能困难。
Transformer用自注意力直接建模任意位置关系。并行、长距离依赖强。计算和数据需求较大。
AE/VAEAE 压缩再重构；VAE 把隐空间变成可采样分布。VAE 可生成，重参数化让采样可训练。AE 随机采样差；VAE 可能生成模糊。
GAN生成器和判别器对抗训练。生成样本清晰。训练不稳定，可能模式崩溃。
Diffusion前向逐步加噪，反向网络学习预测噪声并去噪。训练稳定、生成质量高。采样慢，步骤多。
K-means按距离把样本分到 K 个簇。简单高效。要预设 K，对初值和尺度敏感。
Q-learning学习状态-动作价值函数。model-free，不需知道环境转移模型。大状态空间需函数近似，探索策略重要。

5. 题前补课：先把题目背后的“为什么”点亮

这一页后面是混合题，但你现在不该裸刷。AI 题最散，所以每道题前先有一个可抓住的中心句：题干给什么信号，我就调什么模型；模型解决什么问题，我就写什么第一句。先把下面 6 个小桥读一遍，再进小测。

1. 回归/分类不是看模型名，是看输出 输出连续数值，例如房价、温度、误差大小，先叫回归；输出类别，例如是否垃圾邮件、图片属于猫还是狗，先叫分类。逻辑回归虽然名字里有回归，但它用 sigmoid 输出正类概率，所以常用于二分类。
马上做：“预测学生期末分数”先归回归；“判断学生是否挂科”先归分类。不要先想 SVM 或神经网络。
2. 线性公式要读成中文 看到 X、Y、w、R，不要先怕矩阵。X 是输入表，Y 是标准答案，w 是要学的参数，R 是样本权重。权重大，错这个样本更疼。
例题入口：如果题目问“某样本权重变大有什么影响”，答案不是背闭式解，而是“它在损失里分量更大，模型更在意拟合它”。
3. SVM 图形题先找撑住边界的点 很多直线都能分开两类样本，SVM 选最大间隔那条。离边界最近、贴着间隔的点叫支持向量，它们最可能决定边界。
例题入口：问“删掉哪个点会改变边界”，先找支持向量；远离边界的点通常不是关键。
4. AdaBoost 不看普通错题数，看加权错误 AdaBoost 每轮都会调整样本权重。两个弱分类器都错 2 个样本，也不一定一样好；错高权重样本的那个更糟。
第一行：写 epsilon = 错分样本权重和。然后选加权错误率小且小于 0.5 的弱分类器。
5. 生成模型用训练信号分流 重构输入是 AE；重构 + KL + μ/logσ² 是 VAE；生成器和判别器对抗是 GAN；加噪再去噪、预测噪声是 Diffusion。
马上做：看到“模式崩塌”先想 GAN；看到“预测噪声 epsilon”先想 Diffusion。
6. 强化学习先翻译成四件套 Agent 是做决策者，State 是当前局面，Action 是可选动作，Reward 是即时反馈。Q-learning 用下一状态最大 Q；Sarsa 用实际下一动作。
公式分流：目标项有 max Q(s',a') 多半是 Q-learning；目标项直接写 Q(s',a') 且说实际下一动作，多半是 Sarsa。
这一段怎么用：后面的每道题都先问自己“它落在哪个小桥”。你不用一上来背整章，只要把题干信号接到正确小桥，再写第一句，分数就有落点。

6. 混合题拆题法：一题多信号时先排顺序

AI 综合题最容易崩的地方，不是某个公式完全不会，而是题干同时给了模型名、曲线、表格、场景，你不知道从哪里开始。考场上先别急着回忆所有知识点，先做一个小动作：把题目拆成四层。这样你就算有一个模型没背完整，也不会整题空掉。

1. 任务层先判断它要你做什么：预测连续值、分正负类、找类别结构、压缩表示、生成样本，还是根据奖励选动作。
2. 模型层再锁定模型：回归、SVM、决策树、CNN、Transformer、VAE/GAN/Diffusion、Q-learning 等。
3. 证据层再处理题目给的数据：loss 曲线、混淆矩阵、样本权重、熵表、Q 表、网络结构图。
4. 建议层最后写改进或评价：过拟合怎么治、类别不平衡看什么、生成模型风险、强化学习探索策略。

示范 1：loss 曲线 + 混淆矩阵 + SVM

题干样子：给你训练/验证 loss 曲线，又给混淆矩阵，最后问“该 SVM 模型效果如何，如何改进”。

先别算第一眼先看曲线。训练 loss 低、验证 loss 高，先判断过拟合；两者都高，先判断欠拟合或优化不充分。
再算表混淆矩阵只负责评价指标：TP、FP、FN、TN -> Accuracy、Precision、Recall、F1。类别不平衡时，要特别说明不能只看 Accuracy。
再说模型SVM 的标准话是最大间隔、支持向量、超平面，必要时提核函数。改进可写调 C/核参数、正则化、特征标准化、处理类别不平衡。
答案骨架“从曲线看，该模型存在……；从混淆矩阵看，Precision/Recall 分别为……；由于 SVM 依赖最大间隔和支持向量，可以通过……改善泛化。”

示范 2：图像任务 + CNN / ViT / Transformer

题干样子：让你为图像分类任务选择模型，或解释为什么 ViT 要把图片切成 patch，为什么 Transformer 能建模全局关系。

任务层图像分类仍然是分类任务，输出通常是类别概率；别因为出现图片就忘了“输入、输出、损失、评价”这条主线。
CNN如果强调局部纹理、边缘、卷积核、池化、参数共享，答案优先写 CNN。它擅长从局部到整体提特征。
ViT如果强调 patch、token、位置编码、自注意力，就是 ViT/Transformer 路线。图片被切成小块后，每个 patch 像一句话里的一个词。
简答句“CNN 的归纳偏置更强，适合较小数据下提局部特征；ViT 通过自注意力建模全局关系，但通常更依赖数据规模和训练资源。”

示范 3：生成模型辨析题

题干样子：出现“重构输入”“μ/logσ²”“生成器和判别器”“逐步加噪再去噪”，让你判断模型并解释设计目的。

最强题眼对应模型第一句怎么写
压缩输入再重构AEAE 通过编码器压缩表示，再用解码器重构输入，常用于表示学习和降维。
μ、logσ²、重参数化、KLVAEVAE 把隐变量建成可采样分布，用重参数化让采样过程可训练。
生成器、判别器、真假对抗GANGAN 通过生成器和判别器的对抗训练生成逼真样本。
前向加噪、反向去噪、预测噪声DiffusionDiffusion 把生成过程拆成多步去噪，通常训练网络预测噪声。
不要这样答：不要只写“它们都是生成模型”。考试要的是机制差异：AE 重构，VAE 可采样，GAN 对抗，Diffusion 分步去噪。每个模型都写“为什么这样设计”，才能拿简答分。

拆题小练 1：先分层，不完整作答

06:00

题干同时出现“训练 loss 下降很快、验证 loss 后期上升、混淆矩阵中正类 Recall 很低”。请只写三件事：先判断什么问题？再算/看什么指标？最后给两条改进方向。

答案与解析

先看曲线：训练 loss 下降、验证 loss 后期上升，说明模型在训练集上越来越熟，但泛化变差，典型是过拟合。再看混淆矩阵中的正类 Recall，Recall 低表示真实正类被漏掉很多，不能只用 Accuracy 粉饰效果。改进可以写：数据增强、L2/weight decay、dropout、早停、增加正类样本或类别重加权。

关键这题不是单纯“过拟合题”，还混了类别不平衡/漏报风险。答题顺序要把两个信号都照顾到。

拆题小练 2：生成模型先抓最强题眼

06:00

题干说：模型从标准正态采样 ε，并用均值 μ 和方差参数得到隐变量 z，然后解码生成图像。它更像 AE、VAE、GAN 还是 Diffusion？请写出判断依据和一句设计目的。

答案与解析

更像 VAE。判断依据是题干出现了 ε、μ、方差参数和隐变量 z，这对应 VAE 的重参数化技巧：把随机采样写成“均值 + 标准差 × 噪声”。设计目的是让隐空间可以连续采样，同时让梯度能传回编码器输出的 μ 和 σ，从而训练生成模型。

排除AE 只强调压缩和重构，不强调从分布采样；GAN 强调生成器/判别器对抗；Diffusion 强调多步加噪和去噪。

7. 混合小测：先写题眼，再看答案

做混合题前先分流，不要裸做：

如果你还没看上一节，先回到 混合题拆题法。做题时先圈一个主词：连续值走回归，最大间隔走 SVM，熵/纯度走决策树，错分增权走 AdaBoost，Q/K/V走 Attention，状态-动作-奖励走强化学习。你只要先分对路线，后面哪怕公式不完整，也能拿到解释分和步骤分。

例子：

题干同时给“训练误差/测试误差曲线”和“混淆矩阵”。不要一上来算指标。先读曲线判断泛化问题，再读混淆矩阵算 Precision/Recall/F1，最后结合模型名写原理和改进建议。这就是综合题的分层作答。

题 1：模型分流

05:00

某任务要根据房屋面积、楼层、位置预测房价。应该优先归为回归还是分类？如果用线性模型，损失函数通常是什么？

答案与解析

这是回归任务，因为房价是连续数值。线性回归常用均方误差 MSE，衡量预测值和真实值差距的平方平均。

迁移预测温度、价格、销量通常是回归；预测是否违约、是否垃圾邮件通常是分类。

题 2：SVM 简答

06:00

用 3-4 句话说明 SVM 的最大间隔思想，并解释支持向量是什么。

答案与解析

SVM 寻找一个能把不同类别分开的超平面，并使该超平面到两类最近样本的距离尽可能大。这个距离就是间隔，最大间隔可以提高泛化能力。支持向量是离分界超平面最近、决定间隔位置的训练样本。核函数可以把非线性问题映射到高维空间处理。

题眼最大间隔、支持向量、超平面、核函数。

题 3：信息增益小计算

07:00

父节点熵为 1.0，按某特征划分后，左子节点占 1/2，熵为 0；右子节点占 1/2，熵为 0.8。该特征的信息增益是多少？

答案与解析

加权子节点熵 = 1/2*0 + 1/2*0.8 = 0.4。信息增益 = 1.0 - 0.4 = 0.6。含义是使用该特征划分后，不确定性下降了 0.6。

迁移决策树选信息增益大的特征，因为它让类别更纯。

题 4：Precision / Recall

07:00

某模型预测 30 个样本为正，其中 24 个真为正；真实正样本共有 40 个。求 Precision 和 Recall。

答案与解析

TP=24。预测为正共有 30，所以 Precision=24/30=0.8。真实正样本共有 40，所以 Recall=24/40=0.6。Precision 看预测正例的纯度，Recall 看真实正例被找回多少。

坑不要把分母混掉：Precision 分母是预测正，Recall 分母是真实正。

题 5：BP 与梯度下降

06:00

BP 反向传播和梯度下降分别解决什么问题？二者是什么关系？

答案与解析

BP 反向传播用链式法则把输出层误差逐层传回，计算各层参数对损失函数的梯度。梯度下降根据梯度方向更新参数，使损失下降。二者关系是：BP 提供梯度，梯度下降使用梯度更新权重。

题眼反向传播不是“直接训练好模型”，它核心是高效计算梯度。

题 6：Transformer / Attention

07:00

解释 Attention 中 Q、K、V 的直观含义，并说明 Transformer 相比 RNN 的一个优势。

答案与解析

Q 可以理解为“我要查什么”，K 是每个位置可被匹配的索引，V 是真正要汇总的信息内容。Attention 先用 Q 和 K 计算相关性权重，再对 V 加权求和。Transformer 的自注意力让任意位置直接建立联系，比 RNN 更容易捕捉长距离依赖，也更容易并行训练。

题 7：K-means / PCA 辨析

06:00

K-means 和 PCA 都属于无监督学习。它们分别在解决什么问题？各自的关键词是什么？

答案与解析

K-means 解决聚类问题，把相似样本分到同一簇，关键词是 K、中心点、最近中心、簇内平方误差。PCA 解决降维问题，用少数主成分保留数据主要变化，关键词是方差最大、正交主成分、降维、信息保留。

坑K-means 不需要标签；PCA 的主成分之间应正交/不相关，不是相关越大越好。

题 8：Q-learning 一步更新

07:00

已知 Q(s,a)=4，α=0.5，γ=0.9，即时奖励 r=2，下一状态最大 Q 值为 6。更新后的 Q(s,a) 是多少？

答案与解析

目标值 = r + γ maxQ = 2 + 0.9*6 = 7.4。TD 误差 = 7.4 - 4 = 3.4。新 Q = 4 + 0.5*3.4 = 5.7。

步骤目标值 -> TD 误差 -> 旧值加学习率乘误差。一步更新只改当前 (s,a)。

题 9：GAN 简答

06:00

说明 GAN 中生成器和判别器分别做什么。为什么 GAN 训练可能不稳定？

答案与解析

生成器从随机噪声生成尽量像真实数据的样本；判别器判断输入是真实样本还是生成样本。两者对抗训练，生成器试图骗过判别器，判别器试图区分真假。训练可能不稳定，因为两者目标相互对抗，能力不平衡时可能梯度不稳定或出现模式崩溃。

题 10：综合题分流

08:00

如果一道综合题同时给出模型训练误差/测试误差曲线、混淆矩阵和一个模型名称 SVM，你的答题顺序应该是什么？

答案与解析

先看训练误差和测试误差：若训练误差低、测试误差高，说明过拟合；若两者都高，可能欠拟合。再根据混淆矩阵计算 Accuracy、Precision、Recall、F1，特别注意类别不平衡时不能只看 Accuracy。最后结合 SVM 解释模型思想：最大间隔、支持向量、核函数，并说明可能的调参或正则化方向。

套路综合题先分流：曲线 -> 泛化问题；表格 -> 指标计算；模型名 -> 原理和优缺点。

题 11：训练策略曲线题

08:00

某图像分类模型训练到后期：训练集 loss 很低，验证集 loss 先下降后上升；另一个模型训练集和验证集 loss 都很高。分别判断是什么问题，并各写两条改进建议。

答案与解析

第一种：训练集 loss 很低、验证集 loss 上升，是过拟合。改进可以写：加 L1/L2 正则或 weight decay；数据增强；早停；降低模型复杂度；增加训练数据。第二种：训练集和验证集 loss 都高，说明模型连训练集都没学好，可能是欠拟合或优化不充分。改进可以写：增加模型容量；训练更久；调整学习率；使用更合适的优化器或归一化策略。

题眼PPT 第 6 讲训练策略图的核心就是看训练损失和测试/验证损失的组合，不是看到 loss 就乱写正则化。
迁移如果题目问“数据增强用在哪个集合”，标准答案是训练集；验证集/测试集只做确定性预处理，不能篡改评估。

题 12：生成模型四分流

09:00

给出四段描述：A. 压缩输入再重构输入；B. 编码器输出 μ/logσ²，通过 z = μ + σ·ε 采样；C. 一个网络生成样本，另一个网络判断真假；D. 前向逐步加噪，反向网络预测噪声并去噪。分别对应什么模型？各写一句“为什么这么设计”。

答案与解析

A 是 AE：通过编码器压缩、解码器重构，主要学习低维表示和重构能力，但随机采样生成能力弱。B 是 VAE：输出 μ/logσ²，把隐变量变成连续分布；重参数化把随机性移到 ε 上，使梯度能传回 μ 和 σ。

z=μ+σ·ε
读法：均值决定中心，sigma 决定波动，epsilon 提供随机噪声；这样既能采样，又能训练。

C 是 GAN：生成器和判别器对抗训练，生成器学会造逼真样本；风险是训练不稳定和模式崩塌。D 是 Diffusion/DDPM：把生成拆成多步去噪，训练目标常是预测噪声；优点是训练稳定、质量高，缺点是采样慢。

题眼重构 -> AE；μ/logσ²/KL/重参数化 -> VAE；生成器/判别器 -> GAN；加噪/去噪/预测噪声 -> Diffusion。
迁移这种题不要只报模型名。每个模型都要补“为什么这样设计”：AE 为表示，VAE 为可采样，GAN 为逼真，Diffusion 为稳定分步生成。
