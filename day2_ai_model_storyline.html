0. AI 新词不要散背：先放进“从数据到答案”的流水线

人工智能题看起来名词很多，其实期末机器学习部分经常围绕同一条线展开：给你一些样本，模型根据输入做预测，损失或纯度告诉它错在哪里，训练方法让它变好，最后用指标或简答解释为什么这样选。

x → 模型 fθ(x) → ŷ
L(θ) = 1n n∑i=1 loss(ŷi, yi)

读成中文：输入特征 x，模型给出预测 ŷ；损失 L(θ) 把每个样本的预测错误加起来，训练就是让这个总错误变小。

分类题 输出是类别。题干常出现“判断是否、属于哪类、正负样本、支持向量、决策树”。
回归题 输出是连续数值。题干常出现“预测房价、误差平方和、线性回归、MSE”。
生成题 输出是“造出来的数据”。题干常出现“生成器、判别器、真假样本、GAN”。
今天的记忆原则：不要背“某某算法是一种算法”这种空话。每个名词都要回答三问：它解决什么问题？它用什么信号变好？考场第一句话怎么写？

0.5 新词地基：先把“题目语言”翻成“模型语言”

AI 最折磨人的地方不是单个公式，而是新词密集出现。你可以把新词先分成四层：数据是什么、模型怎么猜、训练怎么改、考试怎么评价。每个新词都先问“它站在哪一层”。

样本、特征、标签 样本是一条训练记录；特征是模型能看到的输入信息；标签是老师给的标准答案。看到“已知训练数据及其类别”，先反应为监督学习。
损失、经验风险、期望风险 损失是一个样本错多少；经验风险是训练集上的平均错误；期望风险是未来真实数据上的错误。训练集好但新数据差，就是经验风险小、期望风险大。
结构风险、正则化 只压低训练错误会背题，结构风险在经验风险外再惩罚模型复杂度。正则化就是给模型复杂度套绳子，让它别为了贴训练集噪声而乱弯。
纯度、熵、信息增益 纯度说一堆样本是不是同一类；熵越高越混；信息增益说“问完这个属性以后，混乱减少多少”。决策树题基本都在这条线上转。
间隔、支持向量、Hinge 损失 SVM 不满足于“分开”，还想让边界离两类最近样本都远一点。支持向量就是这些最近、最关键、撑住边界的样本。
弱分类器、样本权重、加权投票 AdaBoost 像错题本：上一轮错的样本下一轮变重；靠谱的弱分类器投票权更大；最后不是普通多数票，而是加权投票。
结构风险 = 经验风险 + λ · 模型复杂度惩罚

读成中文：不只看训练集错多少，还要惩罚模型太复杂。λ 是松紧带，太小压不住过拟合，太大可能欠拟合。

选择/填空怎么问给一个关键词，让你认模型：类间/类内 -> LDA；最大间隔 -> SVM；信息增益 -> 决策树；错分增权 -> AdaBoost。
计算题怎么问给表格或小样本，让你算信息增益、加权错误率、Precision/Recall 或一步梯度/更新。重点是步骤，不是复杂算术。
简答题怎么问让你解释为什么这样设计：为什么要正则化，为什么信息增益率修正信息增益，为什么 SVM 泛化好，为什么 BP 需要梯度。
综合题怎么问把曲线、表格、模型名混在一起。先分层：曲线判断泛化，表格算指标，模型名写原理和优缺点。
今天这页的过关句：我看到一个新词时，先判断它属于数据、模型、训练、评价哪一层；再写“它解决什么问题、用什么信号、考试第一句话”。这就是从零到会做题的桥。

1. 监督学习主线：为什么参考题总在问“标签、熵、间隔、权重”

监督学习就是“带答案的练习册”。每个样本不仅有特征 x，还有标准答案 y。模型先猜，再根据错得多不多修正自己。Day2 的 LDA、决策树、SVM、AdaBoost、神经网络，大多都在这个框架里。

模型问：用什么规则从 x 到 y？ LDA 用投影方向分开类别；决策树用一串判断问题；SVM 找最大间隔边界；神经网络用多层权重和激活函数组合特征。
训练问：错了怎么改？ 线性/神经网络常用损失函数和梯度下降；决策树用熵和信息增益选划分；AdaBoost 增大错分样本权重；SVM 通过最大间隔约束找边界。
题干信号你该想到第一句话模板
“类间方差大、类内方差小”LDALDA 的目标是找投影方向，使不同类投影尽量分开、同类投影尽量集中。
“熵、信息增益、纯度、根结点”决策树决策树每次选能最大程度降低不确定性的属性进行划分。
“支持向量、最大间隔、超平面”SVMSVM 通过最大化分类间隔来确定判别边界，离边界最近的点叫支持向量。
“弱分类器、错分样本权重增加”AdaBoostAdaBoost 逐轮训练弱分类器，并提高上一轮错分样本的权重。
“误差反传、链式法则、学习率”BP/梯度下降BP 用链式法则把输出误差反向传播，计算梯度并更新权重。
考试四问模板：任何模型名出现时，都按“它处理什么输入/输出 -> 它用什么信号训练或选模型 -> 它的公式或关键词是什么 -> 它和相邻模型怎么区别”来答。这样比背一句定义稳得多。
先问输入输出 LDA/SVM/决策树/AdaBoost 多数服务分类；线性回归服务连续数值；生成模型服务“造数据”；强化学习服务状态-动作-奖励序列。
再问训练信号 回归看误差，分类看标签/边界/概率，决策树看纯度下降，AdaBoost 看错分样本权重，神经网络看损失梯度。

2. 四个经典模型：不是四个孤岛，是四种处理分类的角度

模型它最像在做什么核心信号容易混的对象考场辨析句
LDA 把高维点投影到更容易分类的低维方向。 类间距离大、类内距离小。 PCA、SVM。 LDA 利用标签信息找判别方向；PCA 更偏无监督保留方差；SVM 更偏找最大间隔分类边界。
决策树 每次问一个属性问题，把样本分得越来越纯。 熵、信息增益、信息增益率、基尼系数。 规则死背、随机森林。 单棵树重在“按属性递归划分”；集成树再谈多棵树投票。
SVM 在两类之间放一条最稳的边界。 最大间隔、支持向量、Hinge 损失。 逻辑回归、感知器。 逻辑回归输出概率，SVM 强调最大间隔；支持向量是离边界最近、决定间隔的点。
AdaBoost 把一群“只比乱猜好一点”的弱分类器组合起来。 加权错误率、错分样本权重增加、弱分类器加权投票。 Bagging、普通投票。 AdaBoost 是串行纠错：上一轮错的样本下一轮更重；Bagging 更像并行抽样再投票。
为什么这张表重要：PPT 第4讲把线性回归、逻辑回归、Softmax、LDA、感知器、SVM 放在同一张线性分类模型表里；考试很可能不是孤立问定义，而是让你区分“损失/目标函数/优化方法/适用场景”。

2.1 LDA：先把二维点投到一条好分的线上

它为什么出现：有时原始特征很多，直接分类不直观。LDA 想找一个投影方向，让类别分开得更明显。

J(w) = wTSBw wTSWw

读成中文：分子是类间散度，越大越好；分母是类内散度，越小越好。LDA 要让这个比值尽量大。

题眼：类间方差、类内方差、投影、判别分析。
易错：不要把 LDA 记成“降维就完了”。它是带标签的监督降维/判别方法，目标是更好分类。

2.2 决策树：每次问一个最能减少混乱的问题

它为什么出现：如果分类规则能写成一串“如果……那么……”，决策树很自然。考试常考熵和信息增益，因为它们决定“先问哪个属性”。

H(D) = − K∑k=1 pk log2pk
Gain(D, a) = H(D) − ∑v |Dv||D| H(Dv)

读成中文：熵 H(D) 是当前集合有多混；信息增益是按属性 a 划分后混乱减少了多少。减少越多，越适合放在树上面。

题眼：高熵/低熵、纯度、信息增益、根结点选择。
易错：“高熵”不是“高级有用”，而是“更混、更不纯”。

2.3 SVM：不只分开，还要分得有余量

它为什么出现：很多边界都能把训练样本分开，但考场不能只问“分开了吗”。SVM 关心“哪条边界最稳”，也就是最大化分类间隔。

minw,b 12 ||w||2 ， yi(wTxi + b) ≥ 1

读成中文：在保证每个样本分对、并且离边界有基本余量的前提下，让 ||w|| 尽量小；等价地让间隔尽量大。

题眼：支持向量、最大间隔、分类超平面、判别模型。
易错：支持向量不是所有样本，而是离边界最近、真正卡住间隔的那些样本。

2.4 AdaBoost：让错题越来越显眼

它为什么出现：一个弱分类器可能只比瞎猜好一点。AdaBoost 的想法是：每一轮让模型更关注上一轮做错的样本，最后把多个弱分类器加权投票。

εt = ∑i:ht(xi)≠yi Dt(i)

读成中文：第 t 轮弱分类器错分了哪些样本，就把这些样本当前权重加起来，得到加权错误率 εt。这一步先于计算 α。

αt = 12 ln ( 1 − εtεt )

读成中文：这一轮错误率 ε 越小，说明弱分类器越靠谱，它在最终投票里的权重 α 就越大。

题眼：弱分类器、错分样本权重增加、加权投票、样本分布更新。
易错 1：不是先看普通准确率，而是先算加权错误率。样本权重不同，错一个重样本比错一个轻样本严重。
易错 2：不是“所有错分样本都从此变对”，而是下一轮训练更重视它们。

3. 神经网络主线：不是魔法，是“多层函数 + 误差回传”

神经网络新词很多，但 Day2 只先抓一条主线：前向传播给出预测，损失函数算错多少，反向传播用链式法则算每个权重的责任，梯度下降更新参数。

θ ← θ − η · ∇θL(θ)

读成中文：参数 θ 沿着让损失 L 下降的方向更新；η 是学习率，控制每步迈多大。

新词它解决什么痛点期末常见问法
BP多层网络里每个权重都影响最终错误，BP 用链式法则算“谁该负责多少”。为什么要求可微？为什么叫误差反向传播？
sigmoid把实数压到 0 到 1，可解释为概率；但极大/极小时梯度会接近 0。二分类输出、梯度消失、饱和区。
LSTM普通 RNN 对长序列容易忘前文，LSTM 用门控决定保留、写入、输出哪些信息。长期依赖、遗忘门、输入门、输出门。
GAN想生成逼真数据时，一个网络造假，一个网络鉴别真假，对抗中共同进步。生成器/判别器、交替训练、模式崩塌。
先别急着背深度学习全家桶：Day2 的神经网络只需要抓“前向传播 -> 损失 -> BP -> 梯度下降”这一条链。CNN、Transformer、VAE、GAN 会在后续主线讲；这里出现 GAN 只是为了让参考习题里的生成器/判别器不陌生。
神经网络题第一句话：神经网络通过前向传播产生预测，用损失函数衡量误差，再通过 BP 和梯度下降调整权重；如果题干出现序列长期依赖，再说明 LSTM 用门控缓解普通 RNN 的梯度消失和记忆不足。

4. 题干分流闸门：看到题先判断“它在问哪一层”

AI 考试有选择、填空、计算、简答和综合题。做题第一秒不要背答案，先判断题目在问哪一层。

题目问法对应层你写什么不要写什么
“该算法属于监督/无监督/强化？”任务层看有没有标签、有没有奖励、有没有聚类/降维。不要把所有机器学习都写成监督学习。
“公式中每个量代表什么？”公式层逐个解释 x、y、w/θ、损失、约束或概率。不要只默公式不解释符号。
“为什么这个模型适合/不适合？”模型比较层从任务类型、数据特点、可解释性、过拟合风险答。不要只写“效果好”。
“根据表格/图选择属性或分类器”小计算层写计算顺序，保留关键中间量，不要求复杂算术。不要直接给结论。
“解释 BP/LSTM/GAN 工作过程”机制层按输入、核心部件、训练信号、输出结果分段写。不要堆英文缩写。

5. 小题检验：先写题眼，再展开解析

题 1：题干说“划分后高信息熵”，你该怎么反应？

先写：高熵代表纯还是混？这个属性好不好，还要看什么？

展开参考答案

题眼：高信息熵 = 不确定性高 = 类别混杂 = 纯度低。

完整反应：如果用某个属性划分后仍然高熵，说明这个划分没有把类别分得很纯；但不能直接说该属性绝对无用，还要看划分前后的熵下降，也就是信息增益。

为什么这样答：参考习题喜欢用“高熵”考你是否把熵和纯度方向搞反。记住：熵越高越混，熵越低越纯。

题 2：看到“支持向量”三个字，标准第一句话是什么？

展开参考答案

第一句话：SVM 通过寻找最大间隔分类超平面来进行分类，支持向量是离分类边界最近、决定间隔大小的样本点。

加分展开：如果题目问优点，可以写泛化能力较好、适合小样本高维问题；如果问局限，可以写核函数和参数选择会影响效果，大规模数据训练成本较高。

题 3：AdaBoost 中某个样本被错分，下一轮发生什么？

展开参考答案

答案：被错分样本在下一轮的权重会增加，训练过程会更关注这些难分样本。

为什么：AdaBoost 的核心是把多个弱分类器组合成强分类器。错分样本变重后，下一轮弱分类器会被迫照顾它们，最终用各轮分类器加权投票。

题 4：BP 为什么不能随便使用不可导函数？

展开参考答案

答案：BP 依赖链式法则从输出层向前计算各层参数对损失的梯度。如果某层映射不可导，就无法正常计算梯度，参数更新会受阻。

注意：实际深度学习里有些点不可导但可用次梯度或约定梯度处理；期末概念题通常考的是“BP 需要可微/可求梯度”这条主线。

题 5：同样是神经网络，什么时候该想到 LSTM？

展开参考答案

题干信号：序列、时间步、文本、前后文、长期依赖、普通 RNN 梯度消失。

标准答案：LSTM 用遗忘门、输入门、输出门和记忆单元控制信息保留与更新，从而缓解普通 RNN 难以捕捉长距离依赖的问题。

题 6：LDA、SVM、决策树、AdaBoost 混在选项里，怎么不晕？

题干分别出现以下信号：A. 类内方差小类间方差大；B. 最大化分类间隔；C. 信息增益选择根结点；D. 错分样本下一轮权重增加。请把它们对应到模型，并写出每个模型的一句话理由。

展开参考答案

A -> LDA：类内小、类间大是 Fisher 线性判别分析的核心目标，用标签信息找更适合分类的投影方向。

B -> SVM：最大化分类间隔、支持向量、分类超平面都是 SVM 的典型题眼。

C -> 决策树：信息增益用于衡量某个属性划分后不确定性降低多少，常用于选择根结点或划分属性。

D -> AdaBoost：AdaBoost 会提高上一轮被弱分类器错分样本的权重，让下一轮更关注难样本。

迁移：如果选项里出现 Bagging，它通常不是“错分样本变重”的串行纠错，而是多份采样数据训练多个模型再投票。

题 7：主观题问“如何缓解过拟合”，标准答案怎么组织？

PPT 和复习资料都反复出现“经验风险小、期望风险大”。请先写：过拟合是什么现象？为什么会发生？至少给 4 个缓解方法。

展开参考答案

现象：过拟合是训练集表现很好、测试集或新样本表现较差，也就是经验风险小而期望风险大。

原因：模型复杂度过高、训练数据太少、噪声被记住、训练过久或调参时泄露测试信息，都可能让模型“背训练集”而不是学规律。

缓解：增加训练数据或数据增强；限制模型复杂度；加入 L1/L2 正则化；使用验证集和早停；交叉验证选择超参数；必要时简化模型或使用 Dropout/批归一化等训练策略。

考试写法：不要只写“正则化”。完整答案要把“训练好测试差 -> 过拟合 -> 原因 -> 方法”串起来，才像主观题答案。

题 8：第 4 讲模型对比表怎么考？

题干给四句话：A. 输出概率并用交叉熵；B. 类间大类内小；C. 最大间隔和 Hinge 损失；D. 阶跃函数硬分类且难优化。请分别对应逻辑回归、LDA、SVM、感知器，并写出每个模型和相邻模型的区别。

展开参考答案

A -> 逻辑回归：逻辑回归先算线性分数，再用 sigmoid 输出正类概率，分类常配交叉熵损失。它和 SVM 的区别是：逻辑回归更偏概率输出，SVM 更偏最大间隔。

B -> LDA：LDA 利用标签信息找投影方向，让类间距离大、类内散度小。它和 PCA 的区别是：LDA 是监督判别方向，PCA 是无监督保留方差方向。

C -> SVM：SVM 目标是找最大间隔分类超平面，常联系 Hinge 损失、支持向量、软间隔和核函数。它和感知器的区别是：感知器只追求分对，SVM 还追求边界稳。

D -> 感知器：感知器用阶跃函数硬分类，分段常数函数难以优化；这也是为什么 PPT 会把它作为通向 SVM 的过渡。

迁移：这种题不要背“某模型是什么”就结束，要按 PPT 表格的维度写：输出/激活函数、损失或目标、优化方法、模型关注点。

题 9：信息增益率为什么不是多余名词？

某属性 A 有 2 个取值，属性 B 几乎每个样本一个取值。按信息增益看，B 可能更高。为什么这不一定合理？信息增益率在修正什么？

展开参考答案

核心：信息增益容易偏向取值很多的属性。因为如果一个属性几乎把每个样本都单独分开，分完以后每个子集都很纯，熵会很低，信息增益看起来很大。

为什么不合理：这种划分可能只是把训练样本记住了，泛化能力不好。比如用“学号”划分学生，训练集上每个叶子都纯，但对新学生没有真正分类规律。

信息增益率怎么修正：信息增益率用属性自身的固有值惩罚分支数过多的属性，降低“取值多就占便宜”的问题。

考试第一句：信息增益率用于修正信息增益偏好取值多属性的缺陷，使划分标准更关注真正有泛化价值的属性。

题 10：训练损失和验证损失一起看，别只会喊过拟合

情况一：训练损失高，验证损失也高。情况二：训练损失低，验证损失高。情况三：训练损失降不动，模型效果一直差。分别判断问题，并写改进方向。

展开参考答案

情况一：训练损失和验证损失都高，更像欠拟合。模型连训练集都学不好，可能容量太小、特征不足、训练不够或正则太强。改进可以写增加模型复杂度、训练更久、补特征、减弱过强正则。

情况二：训练损失低但验证损失高，是过拟合。模型把训练集细节学得太死，泛化差。改进可以写数据增强、增加数据、正则化、早停、降低模型复杂度。

情况三：训练损失降不动且效果差，可能是优化问题。改进可以写调整学习率、换优化器、批归一化、动量、自适应学习率、检查初始化或数据预处理。

迁移：第 6 讲的训练策略图不是让你背“过拟合=正则化”一句话，而是让你先看训练损失和验证损失的组合，再决定写欠拟合、过拟合还是优化问题。
