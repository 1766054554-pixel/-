1. 等值演算：把公式一步步变成你想要的样子

等值演算不是“随便变形”。每一步都必须用一个等值公式做理由。你要像写化学方程式一样：每一行和上一行等价，最后得到目标。

工具公式什么时候用
去蕴涵A -> B ≡ ¬A ∨ B看到箭头先消掉，后面才好用 ∧/∨/¬ 化简。
去等价A ↔ B ≡ (A -> B) ∧ (B -> A)等价题先拆成两个方向。
德摩根¬(A ∧ B) ≡ ¬A ∨ ¬B；¬(A ∨ B) ≡ ¬A ∧ ¬B否定压进括号。
双重否定¬¬A ≡ A化简。
分配律A ∨ (B ∧ C) ≡ (A∨B) ∧ (A∨C)化合取范式。
分配律A ∧ (B ∨ C) ≡ (A∧B) ∨ (A∧C)化析取范式。
吸收律A ∨ (A∧B) ≡ A；A ∧ (A∨B) ≡ A遇到“自己已经够了”时消元。
零一律A∨1≡1，A∧0≡0，A∨0≡A，A∧1≡A最后收尾。
考试写法：每一行和上一行等值，旁边写理由；这比只写最后答案稳得多。
F
≡ … （去蕴涵）
≡ … （德摩根律）
≡ … （分配律 / 吸收律）
≡ 1 （零一律）

2. 真值表与主范式：表格不是笨方法，是保底方法

2024 参考卷里直接出现了补真值表、写主析取范式和主合取范式。你只要把规则记牢，这类题可以机械拿分。

主析取范式：找结果为 1 的行

主析取范式由极小项组成。极小项内部用 ∧，每个变量出现一次。行里变量为 1 写原变量，为 0 写否定。

主合取范式：找结果为 0 的行

主合取范式由极大项组成。极大项内部用 ∨，每个变量出现一次。这里是反直觉规则：行里变量为 0 写原变量，为 1 写否定。

pq行号极小项 m_i极大项 M_i
000¬p ∧ ¬qp ∨ q
011¬p ∧ qp ∨ ¬q
102p ∧ ¬q¬p ∨ q
113p ∧ q¬p ∨ ¬q
记忆法：极小项像“精准命中这一行”，所以 1 写原变量、0 写否定；极大项像“排除这一行”，所以规则反过来。

3. 谓词辖域与替换：先画括号，再动变量

谓词题的坑不是符号多，而是量词到底管到哪里。量词只管它后面紧跟的公式范围；如果没有大括号/括号，通常只管最近的合式公式。考试里“填写变元 y 的辖域”“求替换式”就是在考这个。

概念人话考试动作
辖域量词能管住的范围。先给量词后面的公式画括号。
约束变元被某个量词绑定的变量。在辖域里、名字相同，才被绑定。
自由变元没被量词管住的变量。出现在量词外，或名字不匹配。
代换把命题变元整体替换。全局替换所有匹配项。
替换把某个子公式换成等价式或指定公式。看题目指定的子公式，别换错范围。
可代入替换项里的自由变量不会被新量词捕获。若会被捕获，先换名。
例：先把整条公式看成“量词前缀 + 后面的大括号”，再逐层缩小辖域。
∃x ∀z ∃w (R(w) → ∀y Q(y) ∨ P(x,z))
∃x 的辖域：∀z ∃w (R(w) → ∀y Q(y) ∨ P(x,z))
∀z 的辖域：∃w (R(w) → ∀y Q(y) ∨ P(x,z))
∃w 的辖域：(R(w) → ∀y Q(y) ∨ P(x,z))
∀y 的辖域：Q(y)
  • y 只被 ∀y 管住 Q(y)，不会管住后面的 P(x,z)。
  • x 被最外层 ∃x 管住，所以 P(x,z) 里的 x 是约束变元。

4. 自然语言符号化：全称配蕴涵，存在配合取

自然语言符号化看着像语文题，本质是套结构。你先把论域和谓词解释写清楚，再搭量词骨架。

所有 A 都 B
∀x(A(x) → B(x))
全称句常用蕴涵，因为不是 A 的对象不需要满足 B。
存在 A 是 B
∃x(A(x) ∧ B(x))
存在句要找一个对象同时满足 A 和 B。
唯一存在
∃x(P(x) ∧ ∀y(P(y) → y=x))
先存在一个，再说明所有满足者都等于它。
严格单调递减
∀x∀y(x<y → f(x)>f(y))
注意是 x<y 时，函数值反向变大。
2024 参考卷信号：“若函数 f(x) 是严格单调递减的，则 f(x) 有唯一一最大值点”这种题不是要你背中文，而是要你搭出“若...则...”“严格单调递减”“唯一最大值点”三层结构。

5. 证明骨架预热：今天只接到“能写规范开头”

2024 后半有公理证明题，但今天还不完整展开。你先记住：公理证明不是写感想，而是一行公式一行理由。理由只能是前提、公理、已知定理、推理规则或前面步骤。

证明目标起手骨架后续常用动作
证明 Γ ⊨ Q任取一个使 Γ 中所有公式为真的解释/赋值。推出 Q 也为真。
证明 Γ 不可满足假设存在一个解释使 Γ 全真。推出矛盾。
证明 ⊢ A -> B可用演绎定理思路：假设 A，推出 B。再得到 A -> B。
证明 ∀x∀y(...)任取 x、y。用全称引入收束。

6. 题卡：先写过程，再展开答案

题卡 1：复杂度计算

题目：计算公式 ¬P(x) ∨ R(y) -> ∃x(Q(z) ∧ P(x) ∧ R(x)) 的复杂度。约定原子公式复杂度为 0，¬/量词加 1，二元联结词取两边最大值加 1。

查看答案

左边 ¬P(x) 的复杂度是 1，R(y) 是 0，所以 ¬P(x) ∨ R(y) 的复杂度是 max(1,0)+1=2。右边 Q(z)、P(x)、R(x) 都是 0，Q(z)∧P(x) 是 1，再与 R(x) 合取是 2，外面 ∃x 加 1 得 3。最外层是蕴涵，所以总复杂度 max(2,3)+1=4。

题眼复杂度题不要凭感觉数符号，要画语法树：一元加 1，二元取最大加 1。

题卡 2：等值演算化成永真式

题目：用等值演算证明 P -> (P ∧ (Q -> R) ∨ (¬R ∧ P)) 是永真式。

查看答案 
P -> (P ∧ (Q -> R) ∨ (¬R ∧ P))
≡ ¬P ∨ (P ∧ (Q -> R)) ∨ (¬R ∧ P)       （去蕴涵）
≡ ¬P ∨ (P ∧ (¬Q ∨ R)) ∨ (P ∧ ¬R)       （去蕴涵）
≡ ¬P ∨ P ∧ ((¬Q ∨ R) ∨ ¬R)             （提取 P）
≡ ¬P ∨ P ∧ (¬Q ∨ 1)                    （R ∨ ¬R ≡ 1）
≡ ¬P ∨ P ∧ 1                           （A ∨ 1 ≡ 1）
≡ ¬P ∨ P                               （A ∧ 1 ≡ A）
≡ 1                                    （排中律）
关键把含 P 的两项合并：P∧X ∨ P∧Y ≡ P∧(X∨Y)。

题卡 3：主析取/主合取

题目：若二元公式 F(p,q) 的真值表结果在行号 0、3 为 1，在行号 1、2 为 0，写出主析取范式和主合取范式。

查看答案

行号 0 是 p=0,q=0，对应极小项 ¬p∧¬q；行号 3 是 p=1,q=1，对应极小项 p∧q。所以主析取范式为 (¬p∧¬q)∨(p∧q)。

行号 1 是 p=0,q=1，对应极大项 p∨¬q；行号 2 是 p=1,q=0，对应极大项 ¬p∨q。所以主合取范式为 (p∨¬q)∧(¬p∨q)。

记忆主析取看 1 行；主合取看 0 行。极大项规则反过来。

题卡 4：辖域识别

题目：公式 ∃x∀z∃w(R(w) -> ∀yQ(y) ∨ P(x,z)) 中，写出 y 和 x 的辖域。

查看答案

∀y 的辖域是 Q(y)，所以 y 的辖域是 Q(y)。∃x 位于最外层，辖域是 ∀z∃w(R(w) -> ∀yQ(y) ∨ P(x,z))。

坑∀y 不会管到 P(x,z)。量词只管它后面紧跟的合式公式。

题卡 5：替换式

题目：求 ((P∨R) -> (P∨R∧S) -> P∨R)[(P∨R)/(P↔R)]。按“把子公式 P∨R 替换成 P↔R”理解。

查看答案

若按题目指定的完整子公式 P∨R 替换，则外层前件 P∨R 替换为 P↔R，最后的 P∨R 替换为 P↔R。中间 P∨R∧S 若按优先级理解为 P∨(R∧S)，并不完整等于 P∨R，因此不替换。结果可写为 ((P↔R) -> (P∨R∧S) -> (P↔R))。

若老师的括号约定把中间也显式看成 (P∨R)∧S，则中间可替换成 (P↔R)∧S。考试按题面括号和优先级来，必要时说明你的理解。

关键替换要匹配完整子公式，不是看到 P 和 R 挨得近就换。

题卡 6：唯一最大值点符号化骨架

题目：论域为自然数，P(x,y) 表示 x<y，E(x,y) 表示 x=y，f(x) 表示函数值。写出“若 f 严格单调递减，则 f 有唯一一最大值点”的骨架。

查看答案

严格单调递减可写为：∀x∀y(P(x,y) -> P(f(y), f(x)))，即 x<y 时 f(y)<f(x)。唯一最大值点可写为：∃m(∀x( P(f(x), f(m)) ∨ E(f(x), f(m)) ) ∧ ∀n((∀x(P(f(x), f(n)) ∨ E(f(x), f(n)))) -> E(n,m)))。

整句骨架：严格单调递减 -> 唯一最大值点。不同教材对“≤”的表达可能不同，这里用 P(a,b)∨E(a,b) 表示 a≤b。

重点这类题重点是结构：若...则...；严格递减；存在唯一最大值点。不要一开始就被函数符号吓住。

题卡 7：语义蕴含起手

题目：证明 Γ ⊨ Q 的标准起手怎么写？如果要证 Γ 不可满足，标准起手又怎么写？

查看答案

证明 Γ ⊨ Q：任取一个解释/赋值 v，假设 v 使 Γ 中所有公式为真，下面证明 v 也使 Q 为真。证明 Γ 不可满足：反设存在解释/赋值 v 使 Γ 中所有公式为真，推出矛盾，因此 Γ 不可满足。

规范语义证明一定要出现“任取解释/赋值/模型”这样的语言，因为它在讨论所有模型下的真假。

题卡 8：判断题辨析

题目：“命题逻辑合式公式与其对偶式的析取，一定是永真式。”这句话对吗？给出反例。

查看答案

错误。取公式 A=P∧Q，它的对偶式 A*=P∨Q。A∨A*=(P∧Q)∨(P∨Q)≡P∨Q。当 P=0,Q=0 时结果为 0，不是永真式。

技巧判断“一定”类命题，优先找小反例。两个变量、全 0 常常够用。
