0. 怎么用：不要整页硬背，按时间切包

离散最后阶段最怕两件事：一是被证明题吓住，前面定义和范式也不看了；二是只背定义，证明题一写就空白。Day7 的策略是：先拿可模板化分数，再把证明题写出第一句和关键链条。

救火 45 分钟 看卷面地图、七类题眼、公式快背。每类只背“题眼 -> 第一步 -> 常见坑”。
标准 90 分钟 加做混合训练前 5 题：类型判断、主范式、变量辖域、自然语言翻译、语义证明。
冲分 150 分钟 做完全部混合训练，再勾上场闸门；错题回 Day5/Day6 对应页，不开新坑。
考场心法：离散不是“全靠灵感”的数学。很多题是流程题：真值表、主范式、变量识别、前束化、Skolem 化、反例三件套、公理证明骨架。你只要先把题目放进正确盒子，分数就开始回来了。

来源高频信号Day7 对应动作

定义必须背论域、解释、赋值、模型、可满足/永真/永假、逻辑推论、自由/约束变元、公理系统。定义题用“解决什么问题 + 形式表达 + 例子”答。

2. 七类题眼总控：看到题，先放进盒子

这里是 Day7 最重要的一张表。你不需要一上来想完整答案，只要先识别题型，写出第一步，大脑就不会空白。

题干信号它到底在考什么第一步常见坑
命题、合式公式、复杂度、完全集定义边界题。先写定义，再用最小反例排除错误选项。把含自由变量的句子当命题；把“完全集”和“极小完全集”混掉。
真值表、主析取、主合取、数字电路输出范式计算题。先列变量顺序和真值表，结果为 1 写主析取，结果为 0 写主合取。极小项/极大项正负号写反；漏行。
自由变元、约束变元、辖域、改名、代入谓词语法题。先画量词括号范围，再标每个变量出现是自由还是被绑定。同名变量被内层量词遮住；代入时发生变量捕获。
自然语言、唯一、所有、存在、收敛、单调谓词翻译题。先搭骨架：所有/存在/唯一/如果则，再填谓词。“唯一”只写存在，忘了任意两个满足者相等。
前束范式、无存在前束、Skolem 范式公式加工题。改名 -> 去箭头 -> 否定内移 -> 量词前移 -> 母式合取范式 -> Skolem 化。Skolem 函数依赖对象写错。
Γ ⊨ Q、是否成立、给反例、模型/解释语义推论题。成立：反设前提全真结论假；不成立：写论域/解释/前提真/结论假。反例只让结论假，却没说明前提全真。
Γ ⊢ Q、公理方法、MP、UG、可靠性/完备性形式证明/元定理题。先分清 ⊨ 和 ⊢；公理证明先看能否用演绎定理。题目禁止演绎定理还直接套；可靠性/完备性箭头背反。
考场第一句总口诀：范式题先列真值表；语义题先说“前提全真结论假”；反例题先写“取论域 D=...，并定义 x∈D 上的谓词解释”；公理题先说“在形式系统中由公理和 MP/UG 推出”；自然语言题先搭量词骨架。

3. 公式快背：每条都要会读成人话

这些不是摆着好看的符号。每一条都对应一个考试动作。

语义推出 Γ ⊨ Q ⇔ 每个使 Γ 全真的解释/赋值，也使 Q 真

人话：只要前提都真，结论就逃不掉。要推翻它，就找一个前提全真但结论假的世界。

可靠性与完备性 可靠性：Γ ⊢ Q ⇒ Γ ⊨ Q 完备性：Γ ⊨ Q ⇒ Γ ⊢ Q

可靠性是不证明假东西；完备性是不漏掉真东西。

量词否定 ¬∀x P(x) ⇔ ∃x ¬P(x) ¬∃x P(x) ⇔ ∀x ¬P(x)

不是所有都对，等于至少有一个不对；不存在一个对，等于所有都不对。

两个常见不可反向分配 ∃x(Q(x)∧R(x)) ⇒ ∃xQ(x)∧∃xR(x) ∀xQ(x)∨∀xR(x) ⇒ ∀x(Q(x)∨R(x))

反向一般不成立，因为前者要求同一个对象，后者要求同一边全体成立。

主范式 主析取范式：真值表中结果为 1 的行 → 极小项相∨ 主合取范式：真值表中结果为 0 的行 → 极大项相∧

真行写“成因”，假行写“排除条件”。别把 1 行和 0 行反过来。

Skolem 依赖 ∀x ∃y P(x,y) → P(x, f(x)) ∃y ∀x P(x,y) → ∀x P(x, a)

存在量词前面有哪些全称变量，它就依赖谁。没有前面的全称变量，就用常量。

4. 证明第一句生成器：先写能得分的开头

离散证明题最容易卡在第一句。下面这张表专门解决“我不知道怎么开头”。

目标第一句模板下一步
证明 Γ ⊨ Q反设存在解释/赋值 v，使 Γ 中所有公式为真而 Q 为假。逐条使用前提，推出矛盾。
证明 Γ ⊭ Q取论域 D=...，定义每个 x∈D 上的谓词/命题变元取值。验证前提全真、结论假。
证明 ∀xP(x)任取 a∈D。证明 P(a)，最后全称推广。
证明 ∃xP(x)构造一个对象 a∈D。验证 P(a) 成立。
证明 P→Q若允许演绎定理，假设 P 成立，证明 Q。若题目禁止演绎定理，改用公理链。
公理方法证明 Γ ⊢ Q本题在形式系统中证明，需要列出公式序列及证据。每一步写“前提/公理实例/MP/UG/已给定定理”。

5. 混合训练：每题都先写题眼，再看答案

下面这些题不是为了凑数量，而是覆盖 2020/2022/2024 反复出现的考试动作。

题 1｜类型判断：矛盾式等价为什么可能是永真？

判断公式 (p∧¬p) ↔ (q∧¬q) 的类型。

展开答案

答案：永真式。因为 p∧¬p 永远为假，q∧¬q 也永远为假；假 ↔ 假 为真，所以整个公式在任意赋值下都为真。

迁移不要看到矛盾式就直接说全式永假。先看它在更大公式里扮演什么角色。

题 2｜变量辖域：先画括号再答题

公式 ∃x∀z∃w(R(w) → ∀yQ(y) ∨ P(x,z)) 中，变元 y 和 x 的辖域分别是什么？

展开答案

答案思路：∀y 的辖域是紧跟其后的公式 Q(y)。最外层 ∃x 的辖域是整个 ∀z∃w(...) 部分，里面 P(x,z) 中的 x 被它绑定。

注意：量词辖域不是“所有同名字母肉眼出现的地方”，而是由语法括号决定。若出现内层同名量词，内层会遮住外层。

题 3｜主范式：从真值表行写，不要凭感觉化简

设三变量公式 F 在编号 1、4、7 行为真，在其他行为假。写出 F 的主析取范式和主合取范式的写法原则。

展开答案

答案：主析取范式由结果为 1 的行对应极小项相∨，所以用 1、4、7 行；主合取范式由结果为 0 的行对应极大项相∧，所以用其余 0 行。

符号规则：极小项描述“这一行怎么为真”，变量取 1 写原变量，取 0 写否定；极大项描述“排除这一假行”，变量取 0 写原变量，取 1 写否定。

题 4｜自然语言翻译：唯一最大值点

在自然数论域 N 中，用 x∈N 表示 x 属于自然数论域，用 P(x,y): x<y，E(x,y): x=y，f(x) 表示函数值。翻译：“若 f 严格单调递减，则 f 有唯一一最大值点”。

展开参考骨架

一种可写骨架：

[∀x∈N∀y∈N((P(x,y)) → f(y) < f(x))] → ∃m∈N[∀x∈N(f(x) ≤ f(m)) ∧ ∀u∈N∀v∈N((Max(u)∧Max(v)) → E(u,v))]

其中 Max(t) 可展开为 t∈N ∧ ∀x∈N(f(x) ≤ f(t))。严格的一阶逻辑写法也可以把 x∈N 吸收到论域约定里，不必每个量词都写；但你要知道这些变量都在自然数论域里。

题眼“唯一” = 存在一个 + 任意两个满足条件者相等。

题 5｜语义证明：2022 风味前提全真结论假

证明或反驳：P∧Q→R, ¬S, ¬R∨S ⊨ ¬P∨¬Q。

展开证明

证明：反设存在赋值 v 使所有前提为真而结论 ¬P∨¬Q 为假。结论为假说明 P 真且 Q 真，所以 P∧Q 真。由前提 P∧Q→R 得 R 真。又由前提 ¬S 得 S 假；前提 ¬R∨S 要为真，在 S 假时必须 ¬R 真，即 R 假。于是 R 真且 R 假矛盾。因此不存在这样的赋值，推论成立。

得分句“前提全真、结论假”是语义证明的启动按钮。

题 6｜量词反例：∀ 对 ∨ 不能反向拆

判断 ∀x(Q(x)∨R(x)) ⊨ ∀xQ(x)∨∀xR(x) 是否成立。若不成立，给反例。

展开反例

反例：取论域 D={1,2}，其中 1∈D, 2∈D。令 Q(1) 真、Q(2) 假；R(1) 假、R(2) 真。则对每个 x∈D，Q 或 R 至少一个真，所以 ∀x(Q(x)∨R(x)) 为真。但 Q 不是全真，R 也不是全真，所以 ∀xQ(x)∨∀xR(x) 为假。

迁移：这种题的反例套路是“每个对象都有一边真，但没有哪一边全体真”。

题 7｜可靠性/完备性：别把箭头背反

判断：若公理系统不具有可靠性，那么通过扩充公理集合，有可能使其具备可靠性。

展开答案

答案：错误。不可靠意味着系统已经能证明某个语义上不成立的公式。扩充公理集合只会让可证明公式不减少，不能删除已经可证明的错误公式，所以不能靠加公理恢复可靠性。

人话：一个会证明错东西的系统，不能靠“给它更多出发点”变正确。更多只会更强，不会自动更可靠。

题 8｜公理证明红线：题目禁止演绎定理怎么办？

题目要求“用公理方法证明，不可用演绎定理”。你的开头检查项应该写什么？

展开答案

答案：先说明这是形式系统中的推演题，要给出公式序列和证据。每一步只能来自前提、公理模式实例、已给定定理、MP 或 UG 等规则。若题目禁止演绎定理，不能把“假设前件推出后件”直接作为正式证明，只能把它当草稿骨架，再用给定公理/定理链压成形式证明。

保底策略：如果正式公理链来不及完整写，至少写清目标结构、可用定理和 MP 链条，比空白强很多。
