2. 8 个知识盒子：学到什么程度可以刷题

盒子你要真的懂什么考试怎么问第一动作去哪里补
命题与联结词命题能判真/假；p→q 只有 p 真 q 假时假；完全集、对偶式、重言式/永真式不要混。选择判断、真值表、对偶式反例、公理系统概念。先画真值表或给最小反例，例如 P∧Q。Day1、Day7
主范式主析取看成真行，主合取看成假行；变量顺序固定，0/1 写法别反。给最后一列写主析取/主合取，或从公式补真值表。列变量顺序，圈 1 行和 0 行，再写小项/大项。Day3、final48
谓词与辖域量词只管自己的辖域；自由出现和约束出现看“有没有被对应量词管住”。填空：写 x/y 的辖域；判断自由变元；自然语言翻译。先给每个量词补括号，不要只盯变量出现位置。Day3、2024
等值演算与替换替换只替完整出现的子公式；等值演算常用消箭头、德摩根、分配、吸收、凑 A∨¬A。求替换式，证明公式等值于 1，判断某一步是否等值。先圈完整子公式，再一步一步写等值符号。Day3、2025

2.5 每盒上场模板：看到题后怎么动笔

这部分是给你考场用的。表格告诉你“是哪盒”，这里告诉你“第一句话怎么写、变式怎么认”。读的时候不要背整段，抓住每张卡的口令。

盒 1｜命题与联结词

人话：命题逻辑就是把一句话压成真/假，再用 ¬、∧、∨、→ 拼起来。最容易错的是把“对偶”当“否定”。

p→q 只有 p=1, q=0 时为假
  • 题眼：命题、真值表、对偶式、重言式、完全集。
  • 第一动作：先用最小赋值试反例；要证永真再列完整表。
  • 变式：问“是否总相反/是否永真/是否完备”，都先回真值表。
盒 2｜主范式

人话：主析取是把“让公式为真的行”逐行记下来；主合取是把“让公式为假的行”逐行堵掉。

主析取看 1 行：0 写 ¬p，1 写 p；行内 ∧，行间 ∨
  • 题眼：主析取、主合取、成真赋值、最后一列。
  • 第一动作：固定变量顺序，圈 1 行和 0 行。
  • 变式：从真值表写公式、从公式补表，本质都是找 1/0 行。
盒 3｜谓词与辖域

人话：量词像手电筒，只照自己的辖域。变量长得一样，不代表都被同一个量词管住。

(∀xP(x))→Q(x) 与 ∀x(P(x)→Q(x)) 不是一回事
  • 题眼：辖域、自由变元、约束变元、自然语言翻译。
  • 第一动作：给每个量词补括号，再判断变量是否在括号内。
  • 变式：“所有 A 都 B”常写 ∀x(A(x)→B(x))；“有 A 是 B”常写 ∃x(A(x)∧B(x))。
盒 4｜等值演算与替换

人话：等值演算是一步步换成等价形态；替换式不是查找字符串，而是替完整子公式。

A→B ≡ ¬A∨B；¬(A∧B) ≡ ¬A∨¬B
  • 题眼：等值、化简、替换式 F[G/H]、证明等价。
  • 第一动作：先消箭头/等价，再德摩根，再分配或吸收。
  • 变式：只替换完整出现的 H；括号结构不同就不能替。
盒 5｜前束与 Skolem

人话：前束是把量词整理到最前面；Skolem 是把“存在的对象”改成常元或函数。

∀x∃y∀z∃u P(x,y,z,u) ⇒ P(x,f(x),z,g(x,z))
  • 题眼：前束范式、Skolem 范式、量词依赖。
  • 第一动作：变量改名防冲突，再否定内移，再量词前移。
  • 变式：∃ 前没有 ∀，换常元；∃ 前有 ∀，换依赖这些 ∀ 的函数。
盒 6｜语义后承与反例

人话：Γ⊨Q 是说只要前提都真，结论就逃不掉。要推翻它，就抓一只“前提全真、结论假”的反例。

要证 Γ⊨Q：反设 Γ 全真且 Q 假，推出矛盾
  • 题眼：语义推出、是否有效、是否可满足、构造解释。
  • 第一动作：成立写任意解释；不成立写论域、解释、真值。
  • 变式：量词反例常用 D={1,2}，让性质分摊或见证变化。
盒 7｜公理系统

人话：⊢ 是形式推导，不是“我觉得它真”。每一行都要有身份证，尤其题目说不可用演绎定理时。

行来源 = 前提 / 公理实例 / MP / UG / 已给定理
  • 题眼：推演序列、公理证明、MP、可靠性、完全性。
  • 第一动作：先列允许用的规则，再逐行标来源。
  • 变式：可靠性是 ⊢ 到 ⊨；完全性是 ⊨ 到 ⊢。
盒 8｜关系与结构

人话：关系证明别空说“显然”。把定义展开，任取元素，按条件一步一步推。

最大元唯一：b1≤b2 且 b2≤b1，由反对称性得 b1=b2
  • 题眼：偏序、最大元/极大元、上界、函数复合、双射。
  • 第一动作：写定义；唯一性假设两个对象；双射拆单射和满射。
  • 变式：“存在最大元”要给对象；“最大元唯一”要证两个候选相等。
这 8 张卡怎么用：做真题前先扫一遍卡片标题；遇到错题后不要重看整页，只回对应卡片，补“题眼 -> 第一动作 -> 变式提醒”。
