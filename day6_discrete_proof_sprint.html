0.5 三分钟启动流程：完全不会时，先把题变成“可证明对象”

你现在最怕的不是“最后一步不会”，而是题一放上来，脑子先白屏。离散证明的救命动作是：不要先追完整答案，先把题目翻译成失败情形和第一句话。前三分钟只做下面四步。

1. 圈题眼符号

看到 ⊨ 想“真假世界”；看到 ⊢ 想“公理/推理规则”；看到 ⇔ 想“左右是否同真同假”；看到 ∀/∃ 想“任取 a∈D/构造见证 a∈D”。

2. 写失败情形

证明成立时，先想它怎么失败。例如 Γ ⊨ Q 的失败是“Γ 全真但 Q 假”；∀xP(x) 的失败是“找一个 a∈D 让 P(a) 假”。

3. 固定一个世界

命题题固定一个赋值；谓词题固定一个论域和谓词解释；量词证明先固定任意对象 a∈D 或见证对象 a∈D。证明不是飘在空中写，要有“世界”。

4. 写第一句模板

能写第一句就已经不是空白卷：反设存在赋值使前提全真结论假；任取 a∈D；取见证 a∈D；构造反例 D={1,2}……先下笔，再推进。

你卡住的感觉下一步动作不要做什么
我不知道证什么先判断是成立证明、反例、等价、不等价，还是公理推导。不要直接背一段和题无关的定义。
反例造不出来先让结论假，再倒推怎样让前提真；小论域优先用 {1,2}。不要只写“显然不成立”。
量词顺序晕标出哪个对象必须固定，哪个对象可以随 y 变化。不要把 ∃x∀y 和 ∀y∃x 当成一样。
公理证明乱先确认是 ⊢，再看题目是否允许演绎定理。不要把语义证明的“真值解释”直接当形式推导。
一分钟微练：判断 ∀x(P(x)→Q(x)), ∃xP(x) ⊨ ∃xQ(x) 时，第一句怎么写？
展开微练解析

题眼：这是 Γ ⊨ Q，所以可以走语义证明。第一句写：假设在某个论域 D 和解释下，前提 ∀x(P(x)→Q(x)) 和 ∃xP(x) 都为真。

推进：由 ∃xP(x) 真，存在对象 a∈D 使 P(a) 真；由 ∀x(P(x)→Q(x)) 真，P(a)→Q(a) 真；因此 Q(a) 真，于是 ∃xQ(x) 真。这样就证明了前提全真时结论必真。

这题练什么：存在量词给你一个见证对象，全称量词允许你把规则套到这个对象上。以后看到“所有 P 都是 Q；存在 P；所以存在 Q”，就走这条链。

1. 证明目标路由：先看题目要你做哪件事

离散证明不是“脑子里突然冒出答案”。它更像看到路牌选路线：不同目标有不同第一句。下面这张图就是你考场上的启动器。

人话翻译：证明题不是先想“答案是什么”，而是先想“失败长什么样”。例如 Γ ⊨ Q 的失败就是“Γ 全真但 Q 假”；∀x P(x) 的失败就是“找一个对象让 P 假”；等价式的失败就是“找一种解释让左右真值不同”。

2. 六类必会证明套路：把第一句背熟，再学会推进

题干信号零基础人话标准第一句推进动作常见扣分点
∀x P(x) 要证明论域 D 里每个对象都满足 P。 任取 a∈D。 只用 a∈D 的一般性质推出 P(a)，最后由任意性得 ∀xP(x)。 只取 x=0 或某个特殊对象，不能代表全部。
∃x P(x) 只要找到一个满足 P 的对象。 取/构造 a∈D。 验证 P(a) 成立。 只写“显然存在”，没有给见证对象。
A → B 如果 A 成立，B 必须成立。 假设 A 成立。 从 A 和已知条件推出 B。 反过来假设 B，方向错。
Γ ⊨ Q 所有前提同时为真时，结论一定真。 反设存在解释/赋值使 Γ 全真而 Q 假。 逐条利用前提，推出 Q 真或矛盾。 只证明 Q 有时为真；没处理所有前提。
不成立/不等价 找到一个具体世界让结论失败。 构造反例：令论域 D=...，并定义每个 x∈D 上的谓词解释。 检查前提真、结论假；或左右真值不同。 只让结论假，但前提也假，这不是反例。
Γ ⊢ Q 从公理和推理规则推出 Q。 先写目标结构，再列可用公理/定理/MP。 公理模式实例化，配合 MP 或已给定定理链。 题目禁止演绎定理时仍使用，2022 卷会按比例扣分。
别怕形式符号：⊨ 看真假世界，⊢ 看证明系统。⊨ 是“语义上一定推出”，常靠真值、解释、反例；⊢ 是“形式系统里能推导出来”，常靠公理模式和 MP。2020 总复习明确考可靠性/完备性，就是让你会在这两个符号之间来回翻译。

3. 往年题高频坑：这些不是偏题，是老师真的爱考

坑 1：∃ 分配到 ∧ 通常不行

2020 题考过：∃x(Q(x)∧R(x)) 与 ∃xQ(x)∧∃xR(x) 是否等价。右边可以由不同对象分别满足 Q 和 R，左边要求同一个对象同时满足。

∃x(Q(x) ∧ R(x)) ⇒ ∃xQ(x) ∧ ∃xR(x)，反向不一定成立

反例：论域 D={1,2}，其中 1∈D,2∈D；Q 只在 1 真，R 只在 2 真。右边真，左边假。

坑 2：∀ 分配到 ∨ 通常不行

2020 题也考过：∀x(Q(x)∨R(x)) 不一定推出 ∀xQ(x)∨∀xR(x)。每个对象可以“各自有一边真”，不代表同一边对所有对象都真。

∀x(Q(x) ∨ R(x)) ⇏ ∀xQ(x) ∨ ∀xR(x)

反例：论域 D={1,2}，其中 1∈D,2∈D；Q(1)真 Q(2)假，R(1)假 R(2)真。每个对象都有一边真，但没有哪一边全真。

坑 3：量词顺序决定强弱

∃x∀yP(x,y) 比 ∀y∃xP(x,y) 更强。前者是“有一个固定 x 能照顾所有 y”，后者是“每个 y 可以分别找 x”。固定对象当然可以给每个 y 用，但反过来不一定。

∃x∀yP(x,y) ⊨ ∀y∃xP(x,y)

记忆句：先存在再全称，是“一个人打全场”；先全称再存在，是“每一场可换人”。

坑 4：可靠性和完备性别背反

可靠性说“能证明的都是真的”；完备性说“真的都能证明”。2020 简答直接要求用 ⊢ 和 ⊨ 解释，这是送分也可能扣分的概念题。

可靠性：Γ ⊢ α ⇒ Γ ⊨ α 完备性：Γ ⊨ α ⇒ Γ ⊢ α

别只背中文，必须能把箭头方向写对。

背诵优先级：量词否定、约束变元换名、∀ 对 ∧ 可分配、∃ 对 ∨ 可分配、可靠性/完备性、公理 A1/A2/A3 + MP。这些来自“定义必须背”和总复习，不是我随便挑的。

4. 题卡训练：每题先写题眼，再看解析

下面不是为了刷数量，而是为了把往年题常见模式练出手感。每题都先在输入框写：题眼、第一句、下一步。输入会自动保存在本机。

题 1｜2022 语义证明：前提全真，结论假，推出矛盾

用命题逻辑语义方法判断下面逻辑推论是否成立。若成立，给出证明；若不成立，给出反例。

(P ∧ Q) → R，¬S，¬R ∨ S ⊨ ¬P ∨ ¬Q
展开标准证明与丰满解析
题眼符号是 ⊨，所以这是语义推论题。要证明成立，最稳的方法是排除失败情形：假设所有前提都真，而结论假。
第一句反设存在一个赋值，使 (P∧Q)→R、¬S、¬R∨S 都为真，同时 ¬P∨¬Q 为假。
  1. ¬P∨¬Q 为假，说明 ¬P 假且 ¬Q 假，所以 P 真且 Q 真。
  2. 于是 P∧Q 真；又因为 (P∧Q)→R 为真，所以 R 必须为真。
  3. ¬S 为真，所以 S 为假。
  4. ¬R∨S 为真，而 S 已经为假，所以 ¬R 必须为真，即 R 为假。
  5. 得到 R 真且 R 假，矛盾。因此不存在前提全真结论假的赋值，推论成立。

为什么这题不是硬算真值表：四个变元完整真值表有 16 行，考场可以算，但证明题更希望你写出语义链条。真正拿分的句子是“前提全真、结论假”以及每一步由哪个前提推出。

变式识别：只要题干是若干公式 ⊨ 某公式，且公式数量不多，都优先试这套：反设失败情形，然后逐条用前提把矛盾逼出来。

题 2｜2020 反例：∃ 对 ∧ 不能随便拆

判断下式是否成立；如果不成立，给出反例。

∃x(Q(x) ∧ R(x)) ⇔ ∃xQ(x) ∧ ∃xR(x)
展开反例与迁移解析

结论：不等价。左边能推出右边，但右边不能推出左边。

构造反例：令论域 D={1,2}，其中 1∈D,2∈D。设 Q(1) 真、Q(2) 假；R(1) 假、R(2) 真。

  • ∃xQ(x) 为真，因为 x=1∈D 是见证。
  • ∃xR(x) 为真，因为 x=2∈D 是见证。
  • 所以 ∃xQ(x)∧∃xR(x) 为真。
  • 但不存在同一个 x 同时让 Q(x) 和 R(x) 为真，所以 ∃x(Q(x)∧R(x)) 为假。

真正要记：存在量词拆到合取时，会丢失“同一个对象”的要求。这类题不是背反例，而是知道要把 Q 和 R 分配给两个不同对象。

题 3｜2020 反例：∀ 对 ∨ 不能随便拆

判断下式是否成立；如果不成立，给出反例。

∀x(Q(x) ∨ R(x)) ⊨ ∀xQ(x) ∨ ∀xR(x)
展开反例与迁移解析

结论：不成立。

反例：令论域 D={1,2}，其中 1∈D,2∈D。设 Q(1) 真、Q(2) 假；R(1) 假、R(2) 真。

  • 对 x=1，Q(1)∨R(1) 为真；对 x=2，Q(2)∨R(2) 也为真，所以 ∀x(Q(x)∨R(x)) 为真。
  • ∀xQ(x) 为假，因为 Q(2) 假。
  • ∀xR(x) 为假，因为 R(1) 假。
  • 所以 ∀xQ(x)∨∀xR(x) 为假。

人话记忆：“每个人都有饭或水”不等于“所有人都有饭，或所有人都有水”。可能甲有饭、乙有水。

题 4｜量词顺序：一个固定对象 vs 每次可换对象

判断下面推论是否成立，并解释原因。

∃x∀yP(x,y) ⊨ ∀y∃xP(x,y)
展开证明与变式提醒

结论：成立。

证明：假设 ∃x∀yP(x,y) 为真，则存在某个 a∈D，使得对任意 y∈D，P(a,y) 都成立。现在要证明 ∀y∃xP(x,y)：任取 y∈D，由刚才的固定对象 a 可知 P(a,y) 成立，因此对这个 y，存在 x=a∈D 使 P(x,y) 成立。由于 y 是任取的，所以 ∀y∃xP(x,y) 成立。

变式提醒：反过来 ∀y∃xP(x,y) ⊨ ∃x∀yP(x,y) 通常不成立。因为前者允许每个 y 分别找不同的 x，后者要求同一个 x 对所有 y 都行。

题 5｜2022/2020 概念题：可靠性与完备性

用符号 ⊢ 和 ⊨ 解释公理系统的可靠性和完备性，并说明二者方向为什么不能背反。

展开标准答案与考场写法
可靠性：若 Γ ⊢ α，则 Γ ⊨ α 完备性：若 Γ ⊨ α，则 Γ ⊢ α

人话：可靠性保证“系统不会乱证明”：只要形式系统能推导出 α，α 在语义上就确实由 Γ 推出。完备性保证“系统够强”：只要 α 在语义上确实由 Γ 推出，形式系统也能把它证明出来。

为什么方向不能反：可靠性从可证明走向语义真；完备性从语义真走向可证明。一个管“不犯错”，一个管“不漏掉”。

题 6｜公理证明起手：先看题目是否禁止演绎定理

2022 证明题明确写过：用公理方法证明，不可用演绎定理，使用演绎定理按比例减分。遇到这种题，你至少应该先写出什么检查项？

展开作答框架与扣分点

第一步不是直接套自然推理。你要先写清楚：这是 Γ ⊢ Q 的形式推演题，不是 Γ ⊨ Q 的语义证明题；题目若禁止演绎定理，就不能把“假设前件推出后件”直接作为正式证明。

  1. 列出前提 Γ 和目标 Q。
  2. 检查题目给的可用定理，比如传递：P→Q, Q→R ⊢ P→R；自反：⊢Q→Q；逆否类定理。
  3. 把目标拆成能用定理连接的中间箭头。
  4. 每一步标注依据：前提、已给定定理、A1/A2/A3 实例、MP。

考试策略：短时间内不追求每道公理证明都从零原创，但必须会识别禁止演绎定理这类红线。会写自然骨架也比空白强，但正式答案要尽量补依据。

题 4.5｜伪证明诊断：为什么“结论假”还不够？

同学想证明下面推论不成立，只写了：“令 Q(1) 真，R(2) 真，所以不存在同一个 x 使 Q(x) 和 R(x) 都真，结论假。”这个答案为什么不够？应该补哪三件事？

∃xQ(x) ∧ ∃xR(x) ⊭ ∃x(Q(x) ∧ R(x))
展开诊断与标准补法

问题在哪里：反例不是只让结论假。要推翻 Γ ⊨ Q，必须给出一个世界，使前提 Γ 全真，而结论 Q 假。原答案没有完整说明论域，也没有把 R(1)、Q(2) 的真假补齐，所以“没有同一个 x”还不够严谨。

标准补法：令论域 D={1,2}。规定 Q(1) 真、Q(2) 假；R(1) 假、R(2) 真，其中 1∈D, 2∈D。于是 ∃xQ(x) 为真，见证是 1∈D；∃xR(x) 为真，见证是 2∈D，所以前提真。但没有对象同时满足 Q 和 R，因此 ∃x(Q(x)∧R(x)) 为假。

以后看到反例题：默念三件套：论域/解释是什么，前提为什么真，结论为什么假。少任何一件都像把答案写到半路就收摊，考场容易扣分。

5. 错题回炉：把“不会”改写成下次的启动信号

离散错题不要只抄答案。你要把错因写成“下次看到什么就怎么做”。下面这个模板直接照填。

错题复述模板
  1. 这题题眼是：______。
  2. 我第一次错在：把 ______ 当成 ______。
  3. 正确第一句应该是：______。
  4. 下次看到 ______，我先 ______。
常见错因翻译
  • “我不会证明” → 我没有识别目标类型。
  • “反例总造不出来” → 我没写清前提真、结论假。
  • “量词题看不懂” → 我没区分固定对象和可变对象。
  • “公理证明乱” → 我没分清 ⊨ 和 ⊢，也没看能否用演绎定理。
