0.5 参考习题前置补课：五种 AI 方法谱系先认人

你在 Day1 参考习题里碰到“符号主义的逻辑推理、问题求解的探寻搜索、数据驱动的机器学习、行为主义的强化学习、博弈对抗的决策智能”会懵，是正常的，因为它们不是一个个孤立名词，而是 AI 发展里五条常见路线。先把它们当成五种“解题性格”来认。

符号主义：像写严谨证明 核心是知识表示 + 逻辑推理。人把规则、事实、关系写成符号，机器按规则推出结论。题眼常是“知识库、规则、谓词逻辑、推理、专家系统、可解释”。优点是清楚可解释；缺点是规则很难写全，开放世界容易脆。
问题求解/搜索：像在迷宫找路 核心是状态空间 + 操作 + 路径。从初始状态出发，通过动作一步步到目标。题眼常是“状态、节点、路径、启发式、A*、蒙特卡洛树搜索”。它不是先训练模型，而是在可能行动里找更好的走法。
机器学习：像从样本里归纳规律 核心是数据 + 模型 + 损失/指标。给样本和反馈，模型自动调参数。题眼常是“训练集、特征、标签、回归、分类、聚类、过拟合、泛化”。监督学习靠标签，无监督学习靠结构，深度学习是其中一类多层表示学习方法。
行为主义/强化学习：像打游戏学策略 核心是状态 + 动作 + 奖励 + 策略。没有每一步标准答案，环境只给奖励，智能体要最大化长期回报。题眼常是“MDP、reward、policy、value、Q-learning、探索利用”。它和监督学习最大区别是反馈可能延迟、只告诉得分。
博弈对抗/决策智能：像和对手下棋 核心是多方策略 + 对抗 + 搜索/学习结合。AlphaGo 这类题常把深度学习、强化学习、蒙特卡洛树搜索放在一起。题眼常是“对弈、竞争、生成器/判别器、极小极大、MCTS、策略”。如果题目问 AlphaGo，不要只答“神经网络”，要说学习 + 搜索 + 决策。
考场怎么用这张谱系 看到题干先问：它是在按规则推理、找路径、从数据学模型、靠奖励学策略，还是多方对抗决策？这一步能帮你排掉很多看似高级但不对应题眼的选项。
把选择题扩成简答题：如果题目问“比较几类 AI 方法”，你可以按“核心思想、典型题眼、优点、局限、代表例子”五列答。比如符号主义可解释但规则难完备；机器学习适合从数据中学习但依赖数据和泛化；强化学习适合序贯决策但探索成本高。

小练一下：AlphaGo 应该怎么归类？

如果题目问 AlphaGo 体现了哪些 AI 方法，不要只写一个词。先自己写 2-3 个关键词。

展开参考答案

AlphaGo 综合使用了深度神经网络、强化学习和蒙特卡洛树搜索。神经网络用于评估局面和选择动作，强化学习通过自我对弈改进策略，蒙特卡洛树搜索在巨大搜索空间中进行更有效的决策。它不是主要依赖手写规则库逐步逻辑推理。

1. 总故事线：AI 不是单词表，是六种问题的工具箱

你看到一个 AI 题，先别问“这是哪个英文缩写”。先问：输入是什么、输出是什么、答案靠什么信号变好。这三问会把大部分题分到正确模型家族。

经典机器学习更常考小计算和概念辨析：信息增益、SVM 最大间隔、AdaBoost 权重、Precision/Recall。
深度学习更常考结构和原因：CNN 为什么适合图像，LSTM 为什么缓解长期依赖，Transformer 为什么能并行看全局。
强化学习更常考状态、动作、奖励、价值函数和 Q-learning 一步更新。

3. 公式动作表：公式不是装饰，它说明模型怎样变好

你不要把公式当成“要背的一串符号”。每个公式都按四步读：它属于哪个任务，输入输出是什么，它在优化什么，题目会怎么让你用。

线性回归 / MSE
L(w) = (Y−Xw)T(Y−Xw) = n∑i=1 ei2

动作：让预测值 Xw 和真实值 Y 的平方误差最小。题目给连续值预测时优先想到它。

逻辑回归 / 二分类概率
p(y=1|x) = 11 + e−(wTx + b)

动作：把线性打分压到 0-1 概率。题目出现“是否/正负类/概率输出”时想到它。

决策树 / 信息增益
Gain(D, A) = H(D) − ∑v |Dv||D| H(Dv)

动作：选择能让类别最“变纯”的划分属性。小计算题通常要先算划分前熵，再算划分后加权熵。

AdaBoost / 弱分类器权重
αt = 12 ln 1 − εtεt

动作：错误率越小的弱分类器话语权越大；下一轮提高错分样本权重，让模型盯住难样本。

Attention / 缩放点积注意力
Attention(Q,K,V) = softmax ( QKT√dk ) V

动作：Q 和 K 算相关，softmax 变权重，再加权汇总 V；除以 √d_k 是为了数值稳定。

Q-learning / 一步更新
Q(s,a) ← Q(s,a) + α [ r + γmaxa′Q(s′,a′) − Q(s,a) ]

动作：旧估计向“即时奖励 + 下一状态最好价值”靠近。小计算题先找旧 Q、r、γ、下一状态 maxQ。

考场误区：公式题不要只把公式默出来。简答题里至少写一句“这个公式的目标是什么”；计算题里先列“已知量 -> 代入位置 -> 结果”。

4. 新词词典：每个词都按“问题、机制、考法”记

词它解决什么问题机制/特征考试第一句话
LDA分类时希望同类更聚、异类更分。找投影方向，使类内散度小、类间散度大。“LDA 是监督分类/降维方法，核心是最大化类间差异、最小化类内差异。”
SVM分类边界不只要分开，还要离两边样本尽量远。最大间隔；支持向量决定边界；可用核处理非线性。“SVM 寻找最大间隔超平面，支持向量是离边界最近、决定分类面的样本。”
决策树用一串“问问题”的方式分类。根据信息增益/基尼等指标选划分属性。“决策树每次选择最能降低不确定性的属性，形成可解释的分类规则。”
AdaBoost把很多弱分类器组合成强分类器。错分样本下一轮权重升高；错误率低的弱分类器权重大。“AdaBoost 通过重加权训练弱分类器，逐轮关注难样本，最终加权投票。”
BP神经网络参数很多，需要知道每个权重该怎么改。前向算预测和损失，反向用链式法则传播梯度。“BP 是反向传播算法，用误差梯度从输出层向前更新网络参数。”
LSTM普通 RNN 容易忘远处信息。输入门、遗忘门、输出门控制长期记忆单元。“LSTM 通过门控机制缓解梯度消失，适合长期依赖序列。”
CNN图片有局部空间结构，不能把像素完全打散。卷积共享参数抓局部特征，池化降采样和增强鲁棒性。“CNN 利用局部连接和权值共享提取图像空间特征。”
Transformer长序列需要全局依赖，RNN 顺序计算慢。Self-Attention 让每个 token 直接关注其他 token，可并行。“Transformer 的核心是自注意力，用 Q/K/V 建模序列中任意位置的相关性。”
GAN想生成看起来真实的新样本。生成器造假，判别器辨真伪，二者对抗训练。“GAN 由生成器和判别器组成，通过对抗训练学习数据分布。”
K-means无标签数据想自动分组。交替分配样本到最近中心、更新中心。“K-means 通过迭代更新簇中心，使簇内平方误差减小。”
PCA特征维度太高，想保留主要信息。找方差最大的主方向进行投影。“PCA 是无监督降维方法，选择最大方差方向保留主要信息。”
Q-learning智能体不知道环境模型，也想学每个动作长期好不好。用奖励和下一状态最大 Q 值更新当前 Q 值。“Q-learning 是 model-free、off-policy 的强化学习方法，学习状态-动作价值函数。”

5. 题型路由：看到题干，第一反应应该是什么

题干信号先想到第一步别误判成
给连续数值标签，问预测误差/最小二乘线性回归写预测 ŷ = Xw，损失是平方误差。逻辑回归，因为它输出概率类别。
给类别样本，问哪个属性最适合划分决策树/信息增益先算整体熵，再算每个属性划分后的加权熵。SVM，因为 SVM 问间隔和支持向量。
给弱分类器错误率和样本权重AdaBoost先算加权错误率 ε，再算 α 或更新错分样本权重。普通准确率题，因为 AdaBoost 关心加权错误率。
图片、卷积核、池化、特征图CNN解释局部连接、权值共享、池化降采样。Transformer，除非题干说 patch/self-attention。
Q/K/V、softmax、长距离依赖、并行Transformer/Attention把 Q/K/V 解释成查询、索引、内容。普通 RNN，因为 RNN 是顺序递推。
生成器、判别器、真假样本GAN先说明二者对抗：G 造假，D 辨真伪。VAE，因为 VAE 更强调潜变量和重构/KL。
无标签分组、簇中心、距离最近K-means先分配样本到最近中心，再更新中心。PCA，因为 PCA 是降维不是聚类。
状态、动作、奖励、策略、Q 表强化学习/Q-learning列 s、a、r、s′、maxQ，再代入更新式。监督学习，因为这里没有标准标签 y。

6. 综合训练：先写题眼，再看答案

1. 模型分流题

题干：给定一组带标签的病人数据，每个样本有年龄、血压、化验指标，标签为是否患某病。老师问“如果还希望知道哪些特征最能区分患病和未患病，可以考虑哪些模型或指标？”

展开参考答案与迁移

题眼：带标签 + 是否患病 = 监督学习二分类。想知道哪些特征能区分类别，可以想到决策树的信息增益/基尼，或 LDA 的类间/类内分离思想；若重点是分类边界稳健，也可提 SVM 最大间隔。

标准答法：这是监督二分类任务。若要求可解释地选择特征，可用决策树，根据熵或信息增益选择能最大降低类别不确定性的特征；若强调投影后类别分离，可用 LDA，使类间距离大、类内距离小。

别误判：不要写 K-means，因为题干已有标签；不要直接写 GAN，因为不是生成新病人样本。

2. 公式解释题

题干：解释 Attention(Q,K,V)=softmax(QKT/√dk)V 中 Q、K、V 和 √dk 的作用。

展开参考答案与迁移

题眼：Q/K/V + softmax + √dk = Transformer 注意力公式解释。

标准答法：Q 是查询，表示当前位置想找什么；K 是键，像每个位置提供的索引；V 是值，表示实际要被汇总的内容。QKT 得到相关性分数，softmax 把分数变成权重，再用权重加权汇总 V。除以 √dk 是缩放，防止维度大时点积过大，softmax 过尖、训练不稳定。

迁移：如果问 BERT/GPT，先说它们都基于 Transformer；BERT 偏双向理解，GPT 偏自回归生成。

3. 小计算入口题

题干：Q(s,a)=2，学习率 α=0.5，折扣 γ=0.9，执行动作后得到奖励 r=4，下一状态最大 Q 值为 6。求一次 Q-learning 更新后的 Q(s,a)。

展开参考答案与迁移

第一步：写目标值 target = r + γ maxQ(s′,a′) = 4 + 0.9 × 6 = 9.4。

第二步：更新 Q = 旧 Q + α(target − 旧 Q) = 2 + 0.5 × (9.4 − 2) = 5.7。

常见坑：不要把下一状态所有 Q 值相加；Q-learning 用下一状态最大 Q 值。也不要忘记旧 Q 不是直接被 target 替代，而是按学习率靠近 target。

4. 深度学习比较题

题干：为什么图像分类常用 CNN，而机器翻译、文本生成常用 Transformer？

展开参考答案与迁移

标准答法：图像具有局部空间结构，相邻像素组合成边缘、纹理、局部形状，CNN 的卷积核能通过局部连接和权值共享高效提取局部特征，池化还能降低维度并增强平移鲁棒性。文本序列需要建模远距离词之间的依赖，Transformer 用自注意力让每个 token 直接关注其他 token，能捕捉长距离关系，并且比 RNN 更容易并行训练。

迁移：如果问 ViT，就说它把图像切成 patch，当成 token 再用 Transformer 处理；不是说 CNN 失效，而是换了处理图像的建模路线。

5. 生成模型辨析题

题干：简述 AE、VAE、GAN、Diffusion 在“生成样本”这件事上的差别。

展开参考答案与迁移

答题骨架：AE 主要学习编码和解码，通过重构误差把输入压缩再还原，偏表示学习；VAE 在 AE 基础上给潜变量加入概率分布约束，常有重构项和 KL 项，因此能从潜空间采样生成；GAN 由生成器和判别器对抗训练，生成器学习造出能骗过判别器的样本；Diffusion 通过逐步加噪再学习反向去噪，从噪声一步步生成样本。

考试短句：AE 重构，VAE 概率潜空间，GAN 对抗，Diffusion 去噪生成。

6. 参考习题风格综合题

题干：训练集误差很低，测试集误差很高。另一个模型训练集和测试集误差都很高。分别是什么问题？可以怎么处理？

展开参考答案与迁移

第一种：训练集误差低、测试集误差高是过拟合，说明模型记住了训练数据细节但泛化差。可用正则化、早停、数据增强、减少模型复杂度、交叉验证调参等方法。

第二种：训练集和测试集误差都高是欠拟合，说明模型表达能力不足或训练不充分。可增加特征、提高模型复杂度、训练更充分、减少过强正则化。

迁移：如果题干问训练/验证/测试集，记住验证集用于调参，测试集只用于最终评估，不能用测试集反复调参。
