卷面形态高频信号训练入口过关标准
选择/多选 联合索引、ACID/CAP/BASE、授权传播、触发器、关系代数等价、2NF/3NF/BCNF 概念。 Day6 客观题排雷 能说出为什么错项错，例如“普通 SELECT 权限不等于可传播权限”。

SQL 大题 全部/没有/每组最高/插删改/视图/授权。 Day5 混合小测、Day6 半套模拟 能先写题眼和 SQL 骨架，不把全称查询写成普通 JOIN。

2.5 软工题型雷达：正式卷面要先长这样

这块专门回应“为什么没看到软工选择/判断训练”。以后软工不只按画图推进，还要按 6/18 官方五类题型验收。主页只做叫醒，Day5 负责正式题风格训练，Day6 负责上场彩排。

正式题型卷面会怎么问当前训练入口过关标准
单选题 问“不属于”“最适合”“核心任务”，例如不属于软件过程模型、某阶段不该产出什么。 Day5 题 1、题 3A、题 3E、题 3F 能先写类别边界，再排除干扰项，不靠眼熟。
多选题 把同类概念和混入项放一起，例如项目管理对象、黑盒测试方法。 Day5 题 2、题 3C 逐项判断；能说明“判定覆盖/路径覆盖为什么不是黑盒”。
判断题 用“一定、只要、核心任务是”等绝对说法考概念边界，例如故障是否必然导致失效。 Day5 题 3、题 3B、题 3D 能把绝对词改成严谨说法：可能导致、主要关注、通常依据。
简答题 要求定义、区别、关系和例子，例如故障、错误、失效。 Day5 简答模板、Day6 简答闸门 不能只列三个名词；要写传播链和一个业务例子。
综合题 给案例材料，拆需求分析与建模、系统概要设计、详细设计，可能要求 CodeArts Modeling 作图。 Day4 同题干画图、Day5 分层、Day6 案例彩排 先判断层次，再选图；不把所有小问都画成类图。
历史上机内容处理：2020 卷里的 Git、Vue、Flask 选择题只做低优先保温。除非老师再明确提示，否则它们不抢过程模型、需求/设计/测试、UML 综合题这条主线。

3. AI 细化雷达：把“很多名词”变成一条考试路线

这部分是专门回应“AI 零基础跟不上”和“资料会不会漏”的问题。正常 Day2 仍然只学模型故事线和机器学习题卡；如果你觉得名词散、公式不知道怎么用，打开 AI 考试总纲与模型故事线 当导航；如果担心 PPT 覆盖漏项，查 AI 讲次覆盖地图，不把它们当今天新任务。

先按卷面题型分流：AI 已知题型是选择、填空、计算、简答和综合，且无需计算器。所以复习目标不是“背完所有模型名”，而是看到题先判断它要你做哪种动作。
正式题型卷面会怎么问第一动作当前训练入口
选择/填空 给模型名、性质、适用场景或容易混的说法，问概念边界。 先归类：监督/无监督/强化，回归/分类，判别/生成，再逐项排除。 Day5 参考题眼闯关、Day7 题型雷达
计算题 信息熵/信息增益、卷积乘加、Precision/Recall、Q-learning 一步更新、线性回归损失或梯度方向。 先写公式里每个量是什么，再代小数；无需计算器意味着数值通常不会太刁钻。 Day5 公式默写、Day7 必会动作表
简答题 解释模型原理、优缺点、为什么这样设计、和相邻模型有什么区别。 按“定义 -> 原理 -> 优缺点/适用场景 -> 易混点”写，别只写一句模型名。 Day5 模型比较、Day7 终局题卡
综合题 同时给场景、曲线、表格或模型名，让你判断问题、计算指标、解释模型并给建议。 拆四层：任务层、模型层、证据层、建议层；先分流再展开。 Day5 混合题拆法、AI 参考习题覆盖图

5. 题目质量闸门：以后不再用“看起来像题”的水题

好题必须有来源感 题目要贴近参考习题、往年题、PPT 总结或老师公告范围。不能考页面元信息、分值记忆、工具使用说明这类和期末能力无关的东西。
好题必须先铺垫 如果题目涉及 AdaBoost、Q-learning、Skolem、用例 include/extend，前面必须先有概念、流程、题眼和一个可模仿例子。
好解析不能只有一句话 标准解析要包含：题眼、第一反应、完整步骤、为什么其他选项错、下次遇到什么信号该这么做。
好题要能迁移 同一知识点至少要有一种变式：换题干、换表格、换图、换问法。目标是考试稍微拐弯也能反应。
题型合格形式不合格形式
AI 入门题 给有标签/无标签/奖励反馈场景，让你判断学习范式并说理由。 还没讲算法流程就直接问权重怎么更新。
数据库 SQL 题 给表结构和中文需求，要求写题眼、来源表、连接桥、条件和 SQL 骨架。 只问某份资料第几页、第几部分多少分。
软工画图题 给真实场景，要求先圈角色、动作、对象、状态，再决定图类型和元素。 只让背图名，或者放一张图但不解释每条线为什么存在。
离散证明题 先判断目标类型，再写第一句、可用定义/定理、证明骨架和反例可能性。 一上来整张卷，基础概念没补齐就要求独立证明。
