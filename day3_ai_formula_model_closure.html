0.5 先把第2-6讲串成一条线：AI 不是名词堆，是一台学习机器

Day3 最重要的不是把公式表背满，而是看懂它们为什么会排在一起。第2讲从线性回归开始，是因为它最清楚地展示了输入特征 x -> 模型算预测 -> 损失衡量错多少 -> 优化参数的机器学习骨架。第3讲把输出从连续数字改成类别，第4讲继续问“分类边界怎么更稳”，第5讲换成树和集成，第6讲再问“训练出来的模型到底有没有泛化”。

这条主线怎么读

线性回归教你最基本的预测骨架：给 x，算出 ŷ，用 MSE 评价错多少。

逻辑回归/Softmax把连续分数变成概率，所以它虽然有“回归”两个字，考试上要把它归到分类。

LDA/SVM继续处理分类，但问法变成“怎样的边界或投影更好”：LDA 要类间远类内近，SVM 要最大间隔。

决策树/AdaBoost换了一种思路：树用属性问题降低混乱，Boosting 串行训练弱分类器，让错分样本下一轮更重要。

考场最怕的混淆

经验风险小只说明训练题做得好，不说明新题会；期望风险才接近未来样本表现。

正则化不是“让模型更准”的魔法，而是在损失里加复杂度惩罚，逼模型别为了训练集噪声乱弯。

训练策略题要同时看训练损失和验证/测试损失：训练低、验证高 = 过拟合；两边都高可能是欠拟合或优化没做好。

目标函数：你要最小化或最大化的东西。MSE、交叉熵、Hinge 损失、Fisher 准则、信息增益都属于“指挥训练方向”的规则。
参数：模型里要学的量，比如线性模型的 w 和 b。训练不是背样本，而是找一组能让损失变小的参数。
超参数：训练前由人设定或验证集选择的量，比如正则强度 λ、树深度、学习率。它们通常不直接由梯度学出。
泛化：模型做新题的能力。考试看到“训练集好、测试集差”，要立刻说泛化差、过拟合。
Day3 读法：先把这条主线复述一遍，再看公式。你要能说出“这个公式在链路里负责哪一步”，才算真的开始会用它。

1. 公式怎么学：先把公式翻译成人话

你现在最容易掉进一个坑：看见一整张公式表，以为只要背下来就行。但 AI 考试不会只问“把公式抄一遍”，更常问“这个量是什么意思、为什么要这么做、看到题干怎么选它”。所以这页的公式按四层记。

第一层：任务它解决什么问题？回归、分类、聚类、评价、生成、决策？
第二层：符号x、y、w、Q、K、V、TP、FP 分别是什么？
第三层：动作公式是在预测、度量错误、更新参数，还是选择最优划分？
第四层：题眼题干出现什么词时，我要把它叫出来？
你的背诵目标：不是“我能背出公式”，而是“我能看到题干，把公式叫出来，并解释为什么用它”。如果只能背符号，考试换成简答题就会卡。
新词先解决什么问题考试怎么问你第一句话怎么写
经验风险模型在训练集上的平均损失。训练集表现很好但测试集差。经验风险小只能说明训练集拟合好，不能保证泛化好。
期望风险模型在真实数据分布/未来样本上的损失，理论上更想小。为什么只看训练误差不够。考试时把它翻成“新题表现”，和泛化能力绑定。
结构风险在经验风险之外惩罚模型复杂度。正则化为什么能防过拟合。结构风险最小化是在“拟合训练集”和“别太复杂”之间折中。
交叉熵分类任务里惩罚正确类别概率太低。逻辑回归/softmax 为什么不用 MSE。分类输出是概率，交叉熵直接让正确类概率变大。
支持向量离分类边界最近、决定 SVM 超平面的位置。SVM 为什么叫最大间隔。支持向量撑住边界，普通远点对边界影响小。
Q/K/VAttention 里“查什么、拿什么匹配、汇总什么”。解释 Attention 公式。Q 查 K 得到相关性权重，再按权重汇总 V。

线性回归：用一条线猜数字

先想一个很朴素的场景：用面积、楼层、位置预测房价。每个特征给一个权重，最后加起来，就是预测值。

ŷ=wTx+b
读法：把每个特征按权重加起来，再加一个基础偏移。

怎么记：x 是输入信息，w 是每条信息的重要程度，b 是基础偏移，y_hat 是模型猜出来的答案。

题眼：预测连续数值、房价、分数、温度、销量，优先想到回归。问“误差”时常接 MSE。

当场例题：

题干说“根据房屋面积、楼层、位置预测成交价”，你第一句写：这是连续数值预测，优先归为回归任务，可用线性回归把特征按权重组合成预测值。

MSE：错多少，就把差距平方后平均

模型猜错了，要有一个“错题本分数”。MSE 就是把每个样本的预测值和真实值相减，平方，再取平均。

J= 1n n∑i=1 (ŷi−yi)2
读法：把每个样本“预测 - 真实”的差平方后求平均。

怎么记：括号里是“猜的 - 真的”；平方让正负误差都变成惩罚，也会放大大错误。

题眼：连续值预测、最小二乘、平方误差、异常值敏感。

当场例题：

若真实值是 10，预测值是 13，这个样本的平方误差是 9。写解析时不要只写数字，要补一句：平方误差会放大较大的偏差，所以 MSE 对异常值比较敏感。

逻辑回归：名字叫回归，实际做分类

它先像线性回归一样算一个分数 z，再用 sigmoid 把分数压到 0 到 1，解释成“属于正类的概率”。

p= sigmoid(z) = 11+e−z
读法：把任意分数 z 压到 0-1，解释成正类概率。

怎么记：z 很大，p 接近 1；z 很小，p 接近 0；p 超过阈值就判为正类。

题眼：二分类、概率输出、sigmoid、交叉熵。不要被“回归”两个字骗去答连续值预测。

当场例题：

题干问“是否违约/是否垃圾邮件”，即使模型名叫逻辑回归，也要先写：这是二分类，sigmoid 输出正类概率，常用交叉熵训练。

SVM：把边界画得离两边都远一点

分类不只是把样本分开，还要让分界线离最近的样本尽可能远。最近的那几颗样本叫支持向量，因为它们撑住了边界。

yi (wTxi+b) ≥1 , margin= 2||w||
读法：样本不仅要分对，还要离分界面有足够距离；间隔越大越稳。

怎么记：支持向量像边界旁边的“警戒线”，最大间隔就是让警戒线尽量宽。

题眼：最大间隔、支持向量、超平面、核函数、小中规模分类。

当场例题：

如果题目问“移除哪些点会改变分类边界”，先找离分界线最近的点，这些才是支持向量；远处普通样本通常不决定边界。

信息增益：问一个问题后，世界变清楚了多少

决策树每次都在选“最会分裂数据的问题”。熵表示混乱程度，信息增益表示问完这个属性后，混乱减少了多少。

Gain(D,A) = H(D) − ∑v |Dv||D| H(Dv)
读法：划分前的混乱程度，减去按属性 A 划分后的加权混乱程度。

怎么记：增益 = 原来有多乱 - 分完以后还剩多乱。剩得越少，属性越好。

题眼：决策树、熵、不纯度、属性选择、ID3。

当场例题：

父节点熵 1.0，划分后加权熵 0.4，则信息增益 0.6。考试解释要写：这个属性让不确定性下降 0.6，下降越多越适合作为划分属性。

Precision / Recall：一个管误报，一个管漏报

Precision 看“我说是正的里面，有多少是真的”；Recall 看“真正是正的里面，我抓回来多少”。

Precision= TPTP+FP , Recall= TPTP+FN
读法：Precision 怕误报，Recall 怕漏报。

怎么记：P 的分母是“Predicted positive”；R 的分母是“Real positive”。

题眼：垃圾邮件误拦重视 Precision；疾病漏诊危险重视 Recall；类别不平衡不要只看 Accuracy。

当场例题：

模型预测 30 个正例，其中 24 个真为正；真实正例共有 40 个。Precision = 24/30，Recall = 24/40。先写 TP=24，再写两个分母分别是什么。

Attention：用问题去查索引，再汇总内容

Q 是“我想找什么”，K 是“每个位置的索引”，V 是“真正拿来汇总的内容”。先用 Q 和 K 算相关性，再按权重汇总 V。

Attention(Q,K,V) = softmax ( QKT√dk ) V
读法：用 Q 和 K 算相关性，softmax 成权重，再加权汇总 V。

怎么记：像搜索：query 查 key，最后取 value。除以 sqrt(d_k) 是为了分数别太大，softmax 才稳定。

题眼：Transformer、自注意力、长距离依赖、Q/K/V、Encoder/Decoder。

当场例题：

题干问“为什么 Transformer 能处理长距离依赖”，第一句写：自注意力允许任意位置通过 Q/K 匹配直接建立联系，再加权汇总 V，因此不必像 RNN 那样逐步传递。

Q-learning：旧估计向新目标靠近

Q(s,a) 是“在状态 s 做动作 a 的长期价值”。更新时，不是完全推翻旧值，而是让旧值朝“即时奖励 + 下一步最好价值”移动一点。

Qnew = Qold + α [ r + γ maxQ(s',a') − Qold ]
读法：旧估计值朝“即时奖励 + 折扣后的下一状态最好价值”移动 α 这么大一步。

怎么记：括号里是“目标 - 旧值”，学习率 α 决定改多少，折扣因子 γ 决定多看重未来。

题眼：状态、动作、奖励、策略、探索利用、model-free、Q 表一步更新。

当场例题：

Q_old=4，α=0.5，r=2，γ=0.9，下一状态最大 Q=6。目标值是 7.4，新 Q = 4 + 0.5 × (7.4 - 4) = 5.7。步骤永远是目标值 -> TD 误差 -> 旧值更新。

1.5 公式动作翻译器：先问“它要我做什么”

很多 AI 公式看起来吓人，是因为你一眼看到的是符号，不是动作。其实第14讲总结里反复强调的主线很朴素：学习模型、定义损失、优化参数、评价泛化。所以考场上你先别急着背整条公式，先判断题干正在叫你做哪件事。

预测 题眼是“预测房价/销量/分数”。先写线性回归或回归任务，再解释 x 是特征，w 是权重，ŷ 是预测。
惩罚错误 题眼是“损失函数、交叉熵、MSE”。先问输出是连续值还是类别概率：连续值多接 MSE，分类概率多接交叉熵。
选边界 题眼是“最大间隔、支持向量”。先写 SVM：在能分类正确的边界中，选择离最近样本最远的超平面。
选属性 题眼是“熵、信息增益、增益率”。先写总熵，再写划分后加权熵，最后比较不确定性下降多少。
更新价值 题眼是“状态、动作、奖励、Q 表”。先算目标值 r + γ max Q，再让旧 Q 值向目标靠近。

小题 A：λ 小于 0 为什么危险？

题干给出 L2 正则项 λwTw，追问如果 λ < 0 会怎样。你先别背“不能为负”，要写动作：L2 本来是在惩罚权重变大，λ 为负会反过来鼓励权重变大。

看答案骨架

L2 正则化通过惩罚权重平方和来限制模型复杂度，目的是减少过拟合。λ 为正时，权重越大，损失函数增加越多，优化会倾向于选择较小权重。若 λ 为负，权重越大反而会降低目标函数，这会鼓励模型把权重放大，和正则化目标相反，甚至导致优化不稳定。

小题 B：为什么 ID 属性不能只看信息增益？

参考习题里有信息增益率和决策树构建。若一个属性几乎每个样本一个取值，它会让训练集看起来很纯，但可能只是把样本编号背下来了。这就是信息增益率要修正的偏好。

看答案骨架

信息增益衡量按某属性划分后不确定性减少多少。取值特别多的属性容易把样本拆成很小甚至单样本分支，使训练集熵下降很多，信息增益看起来很高，但它未必学到可泛化规律。信息增益率会惩罚分支过多的属性，降低“记住样本编号”式划分的吸引力。

小题 C：逻辑回归、LDA、SVM 都线性，怎么分？

这类题最像老师会出的简答比较。不要写“它们都是分类器”就结束，要抓三种动作：逻辑回归给概率，LDA 找投影，SVM 选最大间隔边界。

看答案骨架

逻辑回归先算线性分数，再用 sigmoid 输出正类概率，关键词是概率输出、交叉熵、二分类。LDA 利用标签信息寻找投影方向，使类间距离尽量大、类内距离尽量小，关键词是 Fisher 准则、类间/类内散度。SVM 寻找最大间隔分类超平面，支持向量决定边界，关键词是最大间隔、支持向量、核函数。

小题 D：Attention 里的 Q 不是 Q-learning 的 Q

AI 最容易痛苦的地方就是同一个字母在不同章节换意思。Attention 的 Q 是 Query，强化学习的 Q 是动作价值函数。遇到 Q 先看上下文：有 K/V/softmax 就是注意力；有状态、动作、奖励、max Q 就是强化学习。

看答案骨架

若题干出现 Q/K/V、softmax、Transformer、自注意力、token 之间相关性，Q 表示 Query，即当前位置想查询什么；K 是 Key，V 是 Value。若题干出现状态 s、动作 a、奖励 r、折扣因子 γ、下一状态最大 Q，Q(s,a) 表示在状态 s 执行动作 a 的长期动作价值。

给今天的你：你不是要一口气变成 AI 专家。Day3 真正要拿下的是：看到题干不慌，能把它放进任务层、模型层、训练层、评价层里，再写出第一句话。这个能力一旦长出来，后面混合题就会突然没那么吓人。

3. 核心公式：这是速查表，不是第一遍学习材料

如果你刚开始学，先看上一节“公式怎么学”；这一节是用来临考复盘和查漏的。每看一行，都要能补出一句人话：它解决什么、每个符号是什么、题目怎么问。

考点公式/表达符号含义考试提醒
线性回归 ŷ=wTx+b x 是特征，w/b 是参数，ŷ 是预测值。 输出连续值；训练目标通常最小化平方误差。

3.5 公式旁边立刻接题：别等到最后才发现不会用

这一节专门解决你说的“公式堆在那里，但我不知道怎么记、怎么做题”。下面每个小剧场都按先讲题眼 -> 再写第一步 -> 再做小题 -> 再看解析来走。它们来自 OCR 后的《人工智能参考习题》和第14讲课程总结的高频信号：SVM 支持向量、L2 正则、信息增益率、AdaBoost、BP 和 loss 曲线。

使用规则：不要一上来点开答案。先在输入框里写 2-4 句：这题考什么、第一步是什么、哪个相邻知识点最容易混。写不出来也没事，写“我卡在……”也比直接看答案有效。

参考习题型SVM 画边界题：为什么支持向量才决定边界

先讲人话：SVM 不是“随便画一条能分开的线”，而是在所有能分开的线里，选一条离两类最近样本都尽量远的线。离线最近的样本就是支持向量，它们像两边把线“撑住”。参考习题里出现过“画分类边界、圈支持向量、移除哪些样本会改变边界”这类问法。

题眼最大间隔、支持向量、分类边界、移除样本是否改变边界。
第一步先找两类样本之间最窄的通道，再找贴着间隔边界的最近样本；不要去看远离边界的普通点。
易混点支持向量不是“所有样本”，也不是“分类正确的样本”，而是离边界最近、决定间隔的样本。
看丰满解析

标准思路是先定位支持向量。SVM 的边界由最大间隔条件决定，真正卡住这条边界的是离超平面最近的正负样本。远离边界的样本即使移动一点，通常不会改变“最近点是谁”，所以分类边界不变；但支持向量被移除或位置改变，最大间隔的约束就会变化，边界可能随之改变。

考试作答别只写“找最近的点”。更稳的写法是：SVM 通过最大化分类间隔确定超平面，间隔边界上的样本称为支持向量，只有这些样本直接决定超平面位置，因此移除支持向量最可能改变分类边界。

公式解释L2 正则化：lambda 不能小于 0 的原因

先讲人话：L2 正则不是“多加一项显得高级”，它是在损失里加入参数大小惩罚，提醒模型别为了训练集噪声把权重调得特别大。第14讲总结和参考习题都强调 L2 的目的：约束权重、防止过拟合。

L= (Y−Xw)T (Y−Xw) +λwTw
读法：前半项管预测误差，后半项惩罚权重过大。lambda 控制惩罚强度。
题眼L2、正则化、权重平方和、过拟合、lambda 小于 0。
第一步先说明 L2 正则的本意：惩罚过大的权重，使模型别过度复杂。
看丰满解析

当 λ > 0 时，wTw 越大，目标函数增加越多，所以优化会倾向于避免过大的权重，这就是“约束模型复杂度”。当 λ < 0 时，权重越大，λwTw 反而越小，优化过程会被鼓励把权重放大，正好和正则化的目的相反。

标准答案可以写成：L2 正则化的目标是通过惩罚权重平方和来降低模型复杂度、防止过拟合；若 λ 为负，该项不再惩罚大权重，而会降低大权重带来的目标值，从而鼓励模型权重增大，不能起到正则化作用，还可能使优化不稳定。

小计算信息增益率：为什么不只看信息增益

先讲人话：信息增益喜欢“分得很细”的属性。比如一个属性几乎每个样本一个取值，训练集上看起来纯度特别高，但它可能只是把每个样本记住了，泛化很差。C4.5 用信息增益率修正这个偏好。

题眼决策树、属性取值很多、信息增益偏高、信息增益率、过拟合。
第一步先写信息增益 = 划分前熵 - 划分后加权熵，再说明“取值多属性可能虚高”。
ID3
主要用信息增益选属性，直觉是让数据变得更纯。
C4.5
用信息增益率，惩罚分支过多，减少取值多属性的偏置。
看丰满解析

ID 这种属性把每个样本分到很小的分支里，训练集熵会急剧下降，信息增益看起来很大。但这种划分没有学到可泛化规律，只是把训练样本拆散甚至记住，容易过拟合。

信息增益率通过引入划分本身的信息量来惩罚分支过多的属性。作答时要写出三步：信息增益偏好取值多的属性；取值多可能导致过拟合或泛化差；信息增益率用于削弱这种偏好，使属性选择更稳。

综合步骤AdaBoost：先算带权错误，不是普通准确率

先讲人话：AdaBoost 的核心不是“找一个最强分类器”，而是一轮一轮训练弱分类器。上一轮分错的样本，下一轮权重变大，逼后面的弱分类器重点照顾难样本。参考习题里出现了“第一轮选哪个弱分类器、权重怎么变、强分类器怎么写”的综合题。

题眼弱分类器、样本权重、错分样本、第一轮迭代、强分类器。
第一步计算每个弱分类器的加权错误率 ε，不是数普通错了几个。
标准流程算 ε -> 选 ε 最小且小于 0.5 的弱分类器 -> 算 α -> 错分增权、正确降权 -> 归一化 -> 加权投票。
看丰满解析

AdaBoost 的训练分布会给不同样本不同权重，因此错误率必须按样本权重求和。错一个高权重样本，比错一个低权重样本更严重，因为高权重样本代表当前模型还没处理好的难点。

第一轮若初始权重相同，普通错误个数和加权错误率可能一致；但从第二轮开始，权重已经变化，必须按加权错误率选弱分类器。考试里如果题干给了样本权重表，第一行就写“ε = 被错分样本权重之和”，这一步比最终小数更重要。

判断/简答BP 和激活函数：为什么“不可导映射”是坑

先讲人话：BP 是反向传播误差并更新参数的方法，它靠链式法则把输出层的错误一路传回前面的权重。如果中间的映射不可导，梯度就没法稳定地往前传，参数更新会出问题。参考习题里就有 BP 描述正误判断。

题眼BP、误差反向传播、隐藏层、链式法则、可导、梯度下降。
第一步先写 BP 的本质：从输出误差出发，按链式法则计算各层参数梯度。
看丰满解析

BP 的核心是沿网络从后往前传播误差信号，并用链式法则计算每一层参数对损失的影响。如果某个神经元映射函数不可导，链式法则在该位置无法正常给出梯度，梯度下降就难以更新对应参数。

考试表达不要只写“BP 要可导”。更完整的句式是：BP 依赖可导的复合函数结构，通过链式法则把输出层误差转化为各层参数梯度，因此激活函数通常需要可导或几乎处处可导，如 sigmoid、tanh、ReLU 等。

读图诊断训练损失和测试损失：别把所有病都叫过拟合

先讲人话：第6讲和第14讲都强调训练策略：看训练损失和测试/验证损失的组合。训练集都学不好，叫欠拟合或优化问题；训练集很好但测试差，才是典型过拟合。参考习题也有随模型复杂度变化的训练误差、测试误差曲线题。

题眼模型复杂度、训练误差、测试误差、过拟合、欠拟合、优化问题、分布差异。
第一步先分别看训练曲线和测试曲线，而不是只看其中一条。
两边都高
模型没学好，可能欠拟合，也可能训练优化没到位。
训练低测试高
典型过拟合，经验风险低但泛化差。

看丰满解析

训练误差随模型复杂度增加通常会下降，因为模型更容易拟合训练集。测试误差先降后升表示一开始模型表达能力变强、泛化改善；复杂度过高后开始记住训练噪声，测试误差上升，这是过拟合。

如果训练误差和测试误差都高，不能直接写过拟合。更可能是欠拟合或优化不足：模型表达能力太弱、特征不足、训练不充分、学习率不合适等。对应改进是提高模型能力、改进特征、调整优化策略；过拟合才主要用正则、早停、降复杂度和数据增强。

4. 模型对比：简答题按维度写

模型/方法一句话适合什么常见坑
线性回归用直线/超平面拟合连续输出。房价、分数、数值预测。不是分类模型。
逻辑回归用 sigmoid 输出类别概率。二分类。名字有回归，但本质是分类。
LDA找投影方向，让类间远、类内近。线性分类/降维。和线性回归不是一回事。
SVM找最大间隔分类超平面。小中规模分类、边界清晰问题。支持向量决定边界，不是所有样本同等重要。
决策树像问问题一样逐步划分数据。可解释分类、表格数据。容易过拟合，常需剪枝/集成。
AdaBoost把多个弱分类器加权组合成强分类器。分类集成。错分样本权重会变大。
CNN用卷积提取局部空间特征。图像、局部模式。不是主要处理长文本依赖。
RNN/LSTM处理序列；LSTM 用门控保留长期信息。文本、时间序列、语音。普通 RNN 有梯度消失问题。
Transformer用自注意力建模任意位置关系。NLP、生成、理解。BERT 偏理解，GPT 偏生成。
GAN生成器和判别器对抗训练。图像生成、数据生成。训练不稳定、模式崩塌。
强化学习智能体与环境交互，通过奖励学习策略。决策、控制、游戏。不是每个样本都有标准标签。

5. 深度、生成、强化：后半简答高频

RNN vs LSTM RNN 用“隐藏状态”保存过去的信息，但长序列反传容易梯度消失。LSTM 加门控和记忆单元，让重要信息可以保留更久。
Transformer 用自注意力让每个 token 看见其它 token，核心公式是 Attention(Q,K,V)。Encoder 偏理解，Decoder 偏生成。
GAN 生成器负责造假，判别器负责识别真假；训练目标是对抗。优点是样本逼真，问题是训练不稳定。
VAE / 扩散模型 VAE 学潜变量分布；扩散模型先加噪再学会逐步去噪。简答题重在“生成过程”和“训练目标”。
强化学习 四件套：状态、动作、奖励、策略。Q-learning 是 model-free，更新 Q(s,a) 的长期价值估计。
探索与利用 利用是选当前最优，探索是尝试未知动作。epsilon-greedy 用小概率探索，避免只盯眼前最优。

6. 题卡：按考试语言练

题 1：公式解释 - 加权线性回归

解释下方加权线性回归闭式解中 X、Y、w、R 的含义，并说明 R 的作用。

w*= (XTRX)−1 XTRY
读法：带权重 R 的最小二乘闭式解，用来求让预测最贴近 Y 的参数 w。
看答案和解析
2 分说明这是线性回归/加权最小二乘的闭式解，用于连续值预测。
3 分X 是特征矩阵，Y 是真实标签/目标值，w 是待学习参数，R 是样本权重矩阵。
2 分R 改变不同样本对损失的贡献，可用于强调重要样本或降低噪声/异常样本影响。

题 2：SVM 简答

不用推导优化问题，解释 SVM 为什么叫“最大间隔”分类器，支持向量是什么。

看答案和解析

SVM 要找一个分类超平面，把两类样本分开，并让离超平面最近的样本到超平面的距离尽可能大。这个距离就是间隔，间隔越大，模型对边界附近扰动越不敏感，通常泛化更好。

支持向量是离分类超平面最近、真正决定边界位置的样本。移动远离边界的普通样本通常不影响超平面，但移动支持向量可能改变分类边界。

题 3：决策树信息增益

某属性划分前数据集熵为 1.0，按该属性划分后两个子集权重分别为 1/2、1/2，熵分别为 0、0.5。信息增益是多少？这个属性好不好？

看答案和解析
2 分加权子熵 = 1/2 × 0 + 1/2 × 0.5 = 0.25。
2 分信息增益 = 1.0 - 0.25 = 0.75。
2 分增益较大，说明划分后不纯度明显降低，是较好的划分属性。

题 4：Precision / Recall

某模型 TP=30，FP=10，FN=20。求 Precision、Recall，并说明如果漏诊代价高应更重视哪个指标。

看答案和解析
2 分Precision = 30/(30+10)=0.75。
2 分Recall = 30/(30+20)=0.60。
2 分漏诊代价高时更重视 Recall，因为 Recall 衡量真实正例被找回多少。

题 5：RNN/LSTM 简答

为什么普通 RNN 难以学习长期依赖？LSTM 用什么思路缓解？

看答案和解析

普通 RNN 训练时误差需要沿很多时间步反向传播，梯度会反复相乘，可能越来越小或越来越大，导致很早之前的信息难以影响当前参数更新，这就是长期依赖问题。

LSTM 引入记忆单元和输入门、遗忘门、输出门。遗忘门决定旧信息保留多少，输入门决定新信息写入多少，输出门决定输出多少，因此能让重要信息跨更长时间传递。

题 6：Transformer / Attention

解释下方 Attention 公式中 Q、K、V 的直观含义，并说明 Transformer 为什么适合文本建模。

Attention(Q,K,V) = softmax( QKT√dk )V
读法：Q 和 K 计算相关性，softmax 变成权重，再按权重汇总 V。
看答案和解析

Q 可以理解成“我要查什么”，K 是“每个位置可被匹配的索引”，V 是“真正要汇总的内容”。QK^T 计算注意力分数，softmax 变成权重，再对 V 加权求和。

Transformer 用自注意力让句子中任意两个位置直接建立联系，不需要像 RNN 那样一步步传递，因此更适合捕捉长距离依赖，也更容易并行训练。

题 7：GAN 简答

说明 GAN 中生成器和判别器分别做什么，并写出 GAN 训练的基本思想。

看答案和解析

生成器接收随机噪声，生成尽可能像真实数据的样本。判别器接收真实样本和生成样本，判断它们是真还是假。

训练时两者对抗：判别器努力提高识别真假能力，生成器努力骗过判别器。理想情况下，生成器生成的样本越来越接近真实数据分布。

题 8：Q-learning 一步更新

已知 Q(s,a)=2，学习率 α=0.5，折扣因子 γ=0.8，即时奖励 r=4，下一状态最大 Q 值为 5。更新后的 Q(s,a) 是多少？

看答案和解析
2 分目标值 = 即时奖励 + 折扣因子 × 下一状态最大 Q = 4 + 0.8 × 5 = 8。
2 分TD 误差 = 目标值 - 旧 Q = 8 - 2 = 6。
2 分新 Q = 2 + 0.5 × 6 = 5。

考试提醒：一步 Q-learning 只更新当前这个 (s,a)，其它格子不变。

题 9：过拟合与结构风险 - 不是只背“加正则”

某分类模型训练集准确率 99%，测试集准确率 70%。老师问：这是什么现象？请用经验风险、期望风险、结构风险解释，并给出 3 种改进方法。

看答案和解析

现象：这是过拟合。模型在训练集上表现极好，但在测试集上明显下降，说明它可能把训练数据里的偶然细节甚至噪声也学进去了。

风险语言：经验风险指训练集上的损失，训练准确率 99% 说明经验风险很小；期望风险指真实分布或新样本上的损失，测试准确率只有 70% 说明期望风险仍然较大。二者差距大，就是泛化能力差。

结构风险：结构风险最小化是在经验风险之外加入模型复杂度惩罚。它不是让训练集拟合越好越好，而是在“训练集拟合”和“模型别太复杂”之间折中。L1/L2 正则、限制模型深度、早停都属于这个思路。

可写方法：加入 L2/L1 正则；早停；降低模型复杂度；增加数据或数据增强；交叉验证选择超参数；剪枝或 dropout。考试答三种即可，但要说明它们为什么能减少对训练集噪声的依赖。

题 10：综合分流 - 看到题干先选模型和第一步

分别判断下面四个题眼最该想到什么模型/公式，并写第一步：A. “类间尽量远，类内尽量近”；B. “最大间隔、支持向量”；C. “正确类别概率太低要被惩罚”；D. “当前 Q 值往即时奖励加未来最好价值靠近”。

看答案和解析
ALDA/Fisher 线性判别分析。第一句：它寻找投影方向，使投影后类间差异尽量大、类内差异尽量小，常用于监督分类或降维。
BSVM。第一句：SVM 寻找能分开样本且间隔最大的超平面，真正决定边界的是离超平面最近的支持向量。
C交叉熵。第一句：分类模型输出概率，交叉熵直接惩罚正确类别概率太低，因此常用于逻辑回归或 softmax 多分类。
DQ-learning。第一步：先算“即时奖励 + 折扣后的下一状态最大 Q”作为目标值，再用学习率 α 让旧 Q 值向目标值靠近。

这题的训练目标：不是让你背四个名词，而是训练“题眼 -> 模型 -> 第一句”的速度。考试里选择题、填空题、简答题、综合题常常只换包装，核心信号不会变。

题 11：第4讲模型对比 - 别把“线性分类”全答成一个东西

老师给出一个二分类任务，并追问：“逻辑回归、LDA、感知器、SVM 都能和线性分类有关，它们的核心区别是什么？”请按“输出/目标、训练准则、考试关键词、常见坑”写一个简答。

看答案和解析

逻辑回归：先算线性分数，再用 sigmoid 输出正类概率，常用交叉熵训练。关键词是概率输出、sigmoid、二分类。它适合需要解释“属于某类的概率”的场景。

LDA：利用标签信息找一个投影方向，让投影后类间尽量远、类内尽量近。关键词是 Fisher 准则、类间散度、类内散度、降维/分类。它不是在直接最大化分类间隔，而是在找有利于区分类别的投影。

感知器：用硬阈值把线性分数变成类别，目标是把样本分对。它的问题是阶跃函数不光滑、难优化，而且如果有很多条线都能分对，它没有告诉你哪条更稳。

SVM：在正确分类的基础上最大化间隔，离边界最近的支持向量决定超平面。关键词是最大间隔、支持向量、Hinge 损失、核函数。它强调边界稳健性，而不强调概率输出。

为什么这题重要：第4讲 PPT 把这些方法放在同一张“优化方法/损失函数/激活函数”对比表里。考试很可能不让你机械背定义，而是问“同样是线性分类，为什么不是同一个模型”。你只要抓住概率、投影、硬分类、最大间隔四个词，就不会乱。

题 12：第6讲训练策略 - 看 loss 曲线别只喊“过拟合”

有三个模型训练结果：A. 训练损失高，验证损失也高；B. 训练损失低，验证损失高；C. 训练损失下降很慢，两条曲线都卡在较高位置。分别判断更可能是什么问题，并给出对应改进。

看答案和解析
A训练损失和验证损失都高，说明模型连训练集都学不好，更像欠拟合。可增加模型复杂度、加入更有效特征、训练更久，或减弱过强正则化。
B训练损失低、验证损失高，是典型过拟合。模型把训练集细节学得太死，泛化差。可用正则化、早停、数据增强、增加数据、降低模型复杂度、剪枝或 dropout。
C训练损失下降慢且整体较高，可能是优化问题或训练策略不合适。可调整学习率、优化器、批归一化、动量、自适应学习率，或检查初始化和数据预处理。

评分关键：这题不是问你背几个改进方法，而是问你能不能从“训练损失/验证损失的组合”判断病因。第6讲训练策略图的核心就是这张诊断表：过拟合用限制复杂度和增强泛化，欠拟合用提升表达能力，优化问题用学习率和优化器等训练手段。

常见错因：看到验证损失高就一律写正则化。正则化主要针对 B；如果 A 里训练损失也高，模型可能根本没学会，再强行加正则会更学不动。
