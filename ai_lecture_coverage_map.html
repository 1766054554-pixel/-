1. 总路线：AI 不是一堆名词，是五段故事

从老师 PPT 和课程总结看，AI 主线可以压成五段：先学机器学习的基本语言，再学经典模型，再学神经网络，再学深度/生成模型，最后学强化学习和信号处理边界。

核心判断：经典机器学习和参考习题最接近期末可考小题/小计算；深度和生成模型更像选择、填空、简答、模型比较；强化学习可能出一步更新或概念分流；信号处理只先做低成本公式边界。

2.5 掌握级别仪表盘：哪些必须会算，哪些会比较就够

第14讲的用词很有用：熟练掌握意味着要能解释、会写步骤、会做小计算；掌握意味着能按题眼调用模板；了解意味着能辨析概念、比较优缺点，不需要大推导。下面这张仪表盘决定后续 AI 时间怎么花。

A. 必须练到会用

机器学习基本语言 + 线性回归/逻辑回归

来源信号：第14讲明确写“熟练掌握机器学习基本概念”，并要求线性回归、对数几率回归、最小二乘、梯度下降的原理、推导和计算。

  • 考场能力：能解释样本、特征、标签、模型、参数、损失、经验风险、期望风险、泛化、过拟合、正则化。
  • 题型：选择/填空考概念边界；简答考风险和正则化；计算题可能考 MSE、最小二乘骨架、梯度下降更新。
  • 第一步：先判断输出是连续值还是类别，再写模型、损失、优化目标。
A. 必须练到会分流

决策树、信息增益、AdaBoost

来源信号：第14讲要求掌握决策树划分度量和构建算法；最后一节课也提醒决策树计算和作业题型要会。参考习题 OCR 明确覆盖信息增益率和 AdaBoost 综合题。

  • 考场能力：会写总熵 -> 条件熵 -> 信息增益/增益率；会解释错分样本权重为什么变大。
  • 题型：小计算、表格综合题、选择题干扰项。
  • 第一步：决策树先算总熵；AdaBoost 先算加权错误率，不看普通错误个数。
B+. 掌握核心思想

LDA / SVM / 集成学习

来源信号：第14讲写 LDA、SVM、集成学习“了解核心思想，不要求推导和计算”。但参考习题里 SVM 支持向量和分类边界很有考试味。

  • 考场能力：LDA 抓类间远类内近；SVM 抓最大间隔和支持向量；集成学习抓弱分类器组合成强分类器。
  • 题型：选择、填空、简答、图形辨析。
  • 第一步：先写模型选择原则，不展开复杂优化推导。
A. 深度学习小计算要会

CNN、自注意力、训练策略

来源信号：第14讲明确要求掌握 CNN 和自注意力机制的基本原理和计算方式，并了解训练策略如何判断欠拟合、过拟合和优化问题。

  • 考场能力：会做小卷积乘加；会解释 Q/K/V 和 √dk 缩放；会看训练损失/验证损失曲线。
  • 题型：小计算、模型结构简答、曲线诊断综合题。
  • 第一步：图片题先看局部窗口和卷积核；attention 题先写 Q 查 K，再加权汇总 V。
B. 会比较，不追推导

PCA、AE、VAE、GAN、Diffusion

来源信号：第14讲写 PCA 不要求推导；生成模型要求掌握相关概念、了解核心思想，不要求推导和计算。

  • 考场能力：PCA 是最大方差正交方向；AE 是编码-解码重构；VAE 是可采样隐变量分布；GAN 是对抗；Diffusion 是加噪/去噪。
  • 题型：选择、填空、简答比较、模型识别。
  • 第一步：先圈训练信号：重构、KL/采样、生成器/判别器、加噪/去噪。
B+. 概念多，要会场景翻译

强化学习：状态动作奖励、V/Q、Bellman、Q-learning

来源信号：第14讲要求掌握强化学习相关概念，并了解基于策略和值方法；参考习题后半覆盖 MDP、策略评估/优化、Q-learning、Sarsa。

  • 考场能力：能把机器人迷宫/游戏题翻译成状态、动作、奖励、策略；会区分 V(s) 和 Q(s,a)；会做一步 Q-learning 更新。
  • 题型：选择/填空、场景简答、小计算。
  • 第一步：先写四件套，再找 target = r + gamma max Q(s',a')。
复习分配结论：AI 后续不是平均用力。机器学习基础、树/集成、CNN/Attention、RL 小计算要练到会动手；生成模型和 BERT/GPT/ViT 更重模型比较和题眼识别。这样既不漏点，也不会把时间浪费在考试不要求的大推导上。

3. 每讲闯关大纲：学什么 -> 怎么考 -> 当场例题

这一段是给后续 Day3+ 和考前回看用的，不是今天额外加量。它把老师 PPT 和参考习题 OCR 里的信号压成可执行闸门：每讲只问三件事，先学什么、考试怎么问、你当场要写哪一步。

第2-3讲：机器学习语言与线性/逻辑回归

先学什么
样本、特征、标签、模型、参数、损失、训练、泛化；回归看连续值，分类看类别；逻辑回归用 sigmoid 把线性分数变成概率。
怎么考
选择/填空考概念边界；简答考“为什么逻辑回归用于分类”；小题考 MSE、交叉熵、过拟合/欠拟合。
接哪页
Day2 故事线先建立坐标；Day3 公式闭环负责公式和模型比较。
当场例题：题干说“预测 PM2.5 数值”和“判断邮件是否垃圾”。先写：前者是回归，输出连续标量，常用 MSE；后者是二分类，输出类别或正类概率，常用 sigmoid/交叉熵。

第4讲：LDA 与 SVM

先学什么
LDA 是利用标签找投影方向，让类间距离大、类内距离小；SVM 是在可分边界中找最大间隔，支持向量决定边界。
怎么考
参考习题里有 SVM 分类边界和支持向量；选择题常把 LDA、SVM、逻辑回归、决策树放在一起让你分辨。
接哪页
Day2 ML 题卡识别题眼；Day3 公式闭环补“最大间隔/支持向量”的标准表述。
当场例题：题干出现“类间尽量远、类内尽量近、找投影方向”，写 LDA；题干出现“最大间隔、离边界最近样本”，写 SVM 和支持向量。

第5讲：决策树与 AdaBoost

先学什么
决策树是按属性逐步提问；信息增益等于划分前熵减去划分后加权熵；AdaBoost 不是普通投票，而是错分样本下一轮权重变大。
怎么考
参考习题 OCR 明确有信息增益率构建决策树；AdaBoost 常考加权错误率、弱分类器权重、样本权重更新。
接哪页
Day2 ML 题卡先做同类小题；Day5 混合小测把树、SVM、指标放在一起分流。
当场例题：总熵 1.0，属性 A 划分后加权熵 0.25，属性 B 为 0.55，先写 Gain(A)=0.75、Gain(B)=0.45，所以选 A。

第6讲：前馈网络、BP、训练策略

先学什么
神经网络是线性变换 + 非线性激活的层叠；BP 用链式法则把输出层误差往前传；训练/验证曲线用来诊断欠拟合、过拟合、优化问题。
怎么考
参考习题有 BP 描述辨析；PPT 强调 ReLU、分段线性、模型训练策略。常见题是“为什么需要非线性激活”和“训练好测试差怎么办”。
接哪页
Day2 NN 题卡补术语；Day3 公式闭环补 loss/过拟合/评价指标。
当场例题：如果不用激活函数，多层线性变换叠起来仍等价于一层线性模型；因此 ReLU/sigmoid 的作用是引入非线性表达能力。

第7-9讲：CNN、Attention、Transformer、BERT/GPT

先学什么
CNN 抓局部空间特征，核心是感受野、卷积核、参数共享、池化；Attention 用 Q/K/V 做相关性加权；BERT 偏理解，GPT 偏生成。
怎么考
PPT 有 CNN 参数、filter、3x3 卷积核、自注意力是否包含位置信息等思考题；考试更可能问机制、优势、模型比较。
接哪页
AI 深度/生成桥接系统讲；学习舱只做题干分流和第一句作答。
当场例题：题干说“图片切成 patch 当作 token 输入 Transformer”，第一句写 ViT；若题干说“局部窗口滑动、参数共享、池化”，第一句写 CNN。

第10-11讲：PCA、AE/VAE、GAN、Diffusion

先学什么
PCA 是线性降维，不要求推导；AE 是压缩-重构；VAE 多了潜变量分布、重参数化和 KL；GAN 是生成器/判别器对抗；Diffusion 是加噪再去噪。
怎么考
第10讲课程目标明确要求理解 AE/PCA 异同、VAE 的重构损失 + KL；生成模型多考比较和题眼识别。
接哪页
Day4 无监督/生成/强化和 AI 深度/生成桥接，后续 Day5 只做混合分流。
当场例题：题干出现 μ/logσ²、z=μ+σ·ε、KL 散度，写 VAE；出现生成器和判别器，写 GAN；出现逐步加噪/预测噪声，写 Diffusion。

第12-13讲：强化学习

先学什么
先翻译 Agent、Environment、State、Action、Reward、Policy；再区分基于策略、基于价值、Actor-Critic；Q-learning 用下一状态最大 Q，Sarsa 用实际下一动作。
怎么考
参考习题 OCR 有 Sarsa、Q-learning、MDP、探索/利用；考试大概率是概念辨析或一步小表格更新，不会让你算大数。
接哪页
Day4 无监督/生成/强化先建四件套；Day5 混合小测练 Q-learning/Sarsa 分流。
当场例题：Q-learning 第一步不是背公式，而是找当前 s,a,r,s'，再取下一状态所有动作中最大的 Q(s',a')。

4. 公式动作表：不是背符号，是知道考场第一步

AI 公式必须可读，也必须知道“什么时候用”。下面这些是后续 Day3+ 和考前混合题要反复调用的动作。

题眼第一步可读公式/表达常见错法
线性回归 / MSE 先说明输入特征、权重、预测值和真实值。
ŷ=wTx+b
MSE=1nn∑i=1(yi−ŷi)2
 只写公式，不解释 x、w、b、ŷ、y 各代表什么。
决策树 / 信息增益 先算划分前总熵，再算按属性划分后的加权子熵。
Gain(D,a)=H(D)−H(D|a)
 把子熵越大当越好；忘记按分支样本数加权。
AdaBoost 先算每个弱分类器的加权错误率。
εt=错分样本∑iDt(i)
 看普通准确率，不看样本权重；忘记错分样本下一轮权重变大。
Attention 先解释 Q/K/V，再说 QK 算相关性。
Attention(Q,K,V)=softmax(QKT√dk)V
 把 Q 误当 Q-learning 的 Q 表；只背公式不解释缩放。
Q-learning 先找当前 Q、即时奖励、下一状态最大 Q。
target=r+γmaxa'Q(s',a')
Q(新)=Q(旧)+α[target−Q(旧)]
 直接把 target 当新 Q；把下一状态所有 Q 相加，而不是取最大。
