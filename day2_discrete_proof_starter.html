0. 今天边界：先学“证明的起跑动作”

离散证明题最吓人的地方，是你看见题目后不知道第一行该写什么。其实第一行不是靠灵感，而是靠题目目标触发动作。目标长什么样，你就启动对应套路。

今天要会 全称、存在、蕴涵、等价、语义蕴含、反例这 6 类题，第一句怎么写。
今天不做 不分析整张 2024 卷，不做复杂公理系统长证明，不追求一次吃完前束范式和 Skolem。
今天产出 你自己的“证明起手表”：看到目标，能先写出不空白、不跑偏的开头。
为什么这一步很关键：期末证明题不是只考你会不会背定义，还考你能不能把定义组织成合规表达。你先把第一段写稳，后面补定理、规则、细节才有地方接。

1. 证明目标决定第一句：先背这张表

你现在看到证明题，先不要慌，也不要一上来乱变形。先圈出“结论长什么样”。下面这张表就是考场起手动作。

目标形状第一句模板人话理解常见坑
证明 ∀x P(x) 任取论域中一个对象 a。 我要证明“每一个”都满足，所以不能只挑一个特殊对象。 不要写“取一个特殊的 a”，要强调任意性。
证明 ∃x P(x) 取/构造 a = ...，下面证明 P(a)。 存在题需要给“见证”。找到一个就够。 只说“显然存在”通常不够，要说明这个对象为什么满足。
证明 A -> B 假设 A 成立，下面证明 B。 我要证明“只要 A，就 B”，所以先进入 A 成立的世界。 不要反过来假设 B；那是证明 B -> A。
证明 A ↔ B 分别证明 A -> B 与 B -> A。 等价就是双向都能推出。 只证明一个方向最多拿一半结构分。
证明 Γ |= Q 设 Γ 中所有前提均为真，证明 Q 必为真。 语义蕴含说的是：没有一种情况能让前提全真而结论假。 不要把 |= 和 |- 混成一回事；|= 是语义真值，|- 是形式推导。
否定一个“必然成立” 构造一个反例：前提真，但结论假。 想推翻“所有情况都成立”，只需要给出一种失败情况。 反例要同时满足前提，不能只让结论假。
今天先背到能脱口而出：全称任取，存在构造，蕴涵假设前件，等价证双向，语义蕴含看“前提全真结论必真”，不成立就造“前提真结论假”的反例。

2. 语义蕴含：证明题里最常见的“真值版反证”

符号 Γ |= Q 读作“前提集合 Γ 语义蕴含 Q”。它不是在问你“有没有背过推理规则”，而是在问：只要前提都是真的，结论是不是一定真。

Γ⊨Q
读法：所有让 Γ 里前提为真的解释/赋值，也必须让 Q 为真。
  • Γ：一组前提，可以有一个，也可以有多个。
  • ⊨：语义蕴含，讨论所有真值赋值/解释下的真假关系。
  • Q：要被前提保证的结论。
  • 成立标准：不存在“Γ 全真但 Q 假”的情况。
第 1 步假设所有前提都真。
第 2 步为了证明必真，可以反设结论假。
第 3 步从前提真和结论假推出冲突。
第 4 步所以不存在“前提全真结论假”，蕴含成立。

如果你要证明“不蕴含”，方向正好相反：不要写长篇大论，直接给一个真值赋值，让所有前提为真、结论为假。

最重要的检查：反例不是“结论假就行”。反例必须同时满足两件事：前提全真，结论假。少一个都不是反例。

3. 主范式保底：从真值表读答案，不靠玄学变形

主析取范式和主合取范式听起来吓人，但考试里可以靠真值表保底。它们的本质是：把一个公式的真假模式写成唯一标准格式。

项目看真值表哪几行怎么写小项怎么把小项连起来
主析取范式 Σ 看公式结果为 1 的行。 极小项：这一行里变量为 1 写原变量，为 0 写否定。 用“或”连接所有极小项。
主合取范式 Π 看公式结果为 0 的行。 极大项：这一行里变量为 0 写原变量，为 1 写否定。 用“与”连接所有极大项。
P↔Q
真值为 1 的行是 00、11；真值为 0 的行是 01、10。
(¬P∧¬Q) ∨ (P∧Q)
主析取范式：看结果为 1 的行，写极小项，再用 ∨ 连接。
(P∨¬Q) ∧ (¬P∨Q)
主合取范式：看结果为 0 的行，写极大项，再用 ∧ 连接。
  • 主析取看 1 行，每一行写一个极小项。
  • 主合取看 0 行，每一行写一个极大项。

这里最容易错的是主合取范式的小项写法：结果为 0 的那一行，变量为 0 写原变量，变量为 1 写否定。它和主析取正好反着来。

4. 谓词翻译：先把中文句式固定住

谓词逻辑比命题逻辑多了对象和量词。考试常见不是让你“创造数学”，而是把中文准确翻译成公式。你先记四个句式。

中文句式公式模板为什么这样写
所有 A 都是 B ∀x(A(x) -> B(x)) 任取一个对象，如果它是 A，那它必须是 B。
存在 A 是 B ∃x(A(x) ∧ B(x)) 要找一个对象，它同时满足 A 和 B。
没有 A 是 B ∀x(A(x) -> ¬B(x))，或 ¬∃x(A(x) ∧ B(x)) 两种写法等价：每个 A 都不是 B；不存在既是 A 又是 B 的对象。
恰有一个 A ∃x(A(x) ∧ ∀y(A(y) -> y = x)) 先说至少有一个 x，再说任何满足 A 的 y 都必须等于这个 x。
全称常配蕴涵，存在常配合取：这不是死规定，但初学阶段非常稳。看到“所有 A 都 B”，先想 ∀ + ->；看到“存在 A 且 B”，先想 ∃ + ∧。

5. 辖域与代入坑：变量不是名字重复就一定是同一个

量词只管理自己的辖域。你可以把它想成程序里的局部变量：同名不一定同作用域。考试如果问自由变元、约束变元、可代入，核心就是看变量有没有被量词“绑住”。

(∃x (P(x,y)) ) → Q(x)
读法：前面 ∃x 只管括号里的 P(x,y)，管不到箭头右边的 Q(x)。
位置身份原因
P(x, y) 里的 x约束变元它在 ∃x 的辖域里，被 ∃x 管住。
P(x, y) 里的 y自由变元没有 ∀y 或 ∃y 管它。
Q(x) 里的 x自由变元它在 ∃x(P(x,y)) 的括号外，不受前面的 ∃x 管。

代入为什么会出事故

如果你用一个带变量的项去替换自由变元，替换后这个变量掉进了同名量词的辖域，就会被错误绑定，原意被改变。

∃y(x<y)
原式：想把自由变元 x 替换成 y。
∃y(y<y)
错误替换：新放进去的 y 掉进 ∃y 的辖域，被错误绑定，原意改变。
安全动作：先改名避让，比如把原式改成 ∃z(x < z)，再替换 x 为 y，得到 ∃z(y < z)。这才保留原意。

6. 六个推理规则：证明中写“理由”的小工具箱

形式证明每一步都要有理由。下面六个规则不用今天全做难题，但要看见就知道它在干什么。

规则形式人话例子
假言推理 MPP, P -> Q ⟹ Q有条件，也满足条件，所以推出结果。若下雨则地湿；下雨；所以地湿。
拒取式 MTP -> Q, ¬Q ⟹ ¬P如果 P 会导致 Q，但 Q 没发生，所以 P 不可能。若开机则灯亮；灯不亮；所以没开机。
假言三段论 HSP -> Q, Q -> R ⟹ P -> R条件可以接力。学会 SQL -> 会查表；会查表 -> 能做题。
析取三段论 DSP ∨ Q, ¬P ⟹ Q两个里面至少一个真，排除一个，剩下的真。去图书馆或教室；不去图书馆；所以去教室。
化简P ∧ Q ⟹ P并且成立，任一部分都成立。会 SQL 且会范式，所以会 SQL。
附加P ⟹ P ∨ Q一个真，则“它或别的”也真。会 SQL，所以会 SQL 或会离散。

你写证明时，旁边可以标：MP、MT、DS、化简。哪怕老师不要求英文缩写，写出中文理由也能让答案更像“证明”而不是“感觉”。

7. 题卡训练：先写骨架，再看完整解析

每题都先写“题眼 + 第一动作 + 推理路线”。不要只看答案。

题 1：语义蕴含证明

证明：P ∧ Q -> R，¬S，¬R ∨ S |= ¬P ∨ ¬Q。

看标准骨架和解析

题眼：符号是 |=，所以这是语义蕴含题。核心检查“是否存在前提全真但结论假”的情况。

证明骨架：假设三个前提均为真，并反设结论 ¬P ∨ ¬Q 为假。析取为假说明 ¬P 假且 ¬Q 假，因此 P 真且 Q 真，所以 P ∧ Q 真。由 P ∧ Q -> R 为真且 P ∧ Q 为真，得 R 真。又由 ¬S 为真，得 S 假。前提 ¬R ∨ S 为真，而 S 假，所以 ¬R 必真，即 R 假。于是 R 真且 R 假矛盾。因此不存在前提全真而结论假的赋值，原语义蕴含成立。

为什么这题适合入门：它把语义蕴含、反设结论假、蕴涵使用、析取排除都串起来，但每一步都只是基础真值规则。

容易错：不要一上来把 ¬P ∨ ¬Q 当成 P ∧ Q。正确关系是：结论为假时，才推出 P ∧ Q 为真。

题 2：编号式证明

已知 P -> Q，¬Q，P ∨ R，证明 R。

看标准答案和解析 
1. P -> Q          前提
2. ¬Q             前提
3. ¬P             由 1,2 拒取式 MT
4. P ∨ R           前提
5. R              由 3,4 析取三段论 DS

题眼：题干给了 P -> Q 和 ¬Q，这几乎是在提醒你用拒取式 MT 得到 ¬P。再看到 P ∨ R 和 ¬P，就用析取三段论推出 R。

考试写法：每行写公式，右边写“前提”或“由第几行和什么规则得到”。这比散文式证明更稳。

迁移：以后见到 A -> B 与 ¬B，优先想到 ¬A；见到 A ∨ B 与 ¬A，优先想到 B。

题 3：构造反例

判断 P ∨ Q |= P 是否成立；如果不成立，给出反例。

看答案和解析

答案：不成立。取 P = 0，Q = 1，则前提 P ∨ Q = 1，但结论 P = 0。所以存在前提真、结论假的赋值，P ∨ Q 不语义蕴含 P。

题眼：不成立题不要证明半天，直接找反例。前提是一个“或”，想让它真很容易；结论 P 要假，就设 P = 0，再用 Q = 1 保住前提为真。

容易错：如果你取 P = 0，Q = 0，结论虽然假，但前提也假，这不是合格反例。

题 4：主范式小题

求 P ↔ Q 的主析取范式和主合取范式。

看答案和解析

真值表思路：P ↔ Q 在 P、Q 同真或同假时为真，所以 1 行是 00 和 11；0 行是 01 和 10。

主析取范式：(¬P ∧ ¬Q) ∨ (P ∧ Q)。结果为 1 的行写极小项，00 对应 ¬P ∧ ¬Q，11 对应 P ∧ Q。

主合取范式：(P ∨ ¬Q) ∧ (¬P ∨ Q)。结果为 0 的行写极大项，01 对应 P ∨ ¬Q，10 对应 ¬P ∨ Q。

记忆钩子：主析取看 1 行，主合取看 0 行；析取小项“照着写”，合取小项“反着写”。

题 5：谓词翻译

设 Student(x)：x 是学生；Course(y)：y 是课程；Take(x,y)：x 选修 y。翻译：“每个学生至少选修一门课程”。

看答案和解析

答案：∀x(Student(x) -> ∃y(Course(y) ∧ Take(x,y)))。

题眼：“每个学生”控制 x，所以最外层是 ∀x；“至少一门课程”是在每个学生确定后，为这个学生找一门课程，所以 ∃y 放在蕴涵后件里。

为什么不是 ∃y∀x：∃y∀x 会变成“有一门固定课程，所有学生都选它”，意思变强了，和原句不同。

为什么 Student(x) 后面用 ->：论域里可能有非学生对象。全称句“所有学生都...”通常写成 Student(x) -> 后面的性质。

题 6：证明拼图

目标：证明 A ∧ B -> B。请把下面四步按正确顺序写出来，并解释每一步为什么能接上。

  1. 因此 B 成立。
  2. 由 A ∧ B 的含义可知 B 成立。
  3. 假设 A ∧ B 成立。
  4. 所以 A ∧ B -> B 成立。
看答案和解析

答案顺序：3 -> 2 -> 1 -> 4。

解析：目标是蕴涵 A ∧ B -> B，所以第一步假设前件 A ∧ B 成立。根据合取的含义，A ∧ B 成立说明 A 成立并且 B 成立，因此可以取出 B。于是从假设 A ∧ B 出发推出了 B，最后收束为 A ∧ B -> B 成立。

你要学会的不是这题本身：它在训练“蕴涵证明的开头和收尾”。复杂证明的第一步和最后一步常常就是这么来的。
